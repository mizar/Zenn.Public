## 概要

本稿では、整数除算 $\left\lfloor\frac{x}{D}\right\rfloor$ を $\left\lfloor\frac{\lfloor AB/D\rfloor\left\lfloor x/A\right\rfloor}{B}\right\rfloor$ で近似したときの誤差関数 $\Delta(D,A,B,x)=\left\lfloor\frac{x}{D}\right\rfloor-\left\lfloor\frac{\lfloor AB/D\rfloor\left\lfloor x/A\right\rfloor}{B}\right\rfloor$ について、条件 $K<\Delta(D,A,B,x)$ を満たす最小の非負整数 $x$ が存在する場合のその値を求める問題を扱う。不等式変形・存在条件導出・最大値評価に基づく二分探索を組み合わせ、多倍長整数入力に対応した実装可能な解法と計算量評価を与える。本稿では、2025年12月に出題した競技プログラミング問題 [yukicoder no.3398 Accuracy of Integer Division Approximate Function 2](https://yukicoder.me/problems/no/3398) の解析を、関連問題 [yukicoder no.3397 Max Weighted Floor of Linear](https://yukicoder.me/problems/no/3397) の枠組みと併せて整理する。

## はじめに

本稿では、整数除算近似で生じる誤差関数 $\Delta(D,A,B,x)$ に対し、閾値 $K$ を初めて超える最小入力 $x$ を求める問題 「Accuracy of Integer Division Approximate Function 2」 (`divapprox2`) を扱う。背景となる具体例は問題設定の[背景説明](#背景説明)で述べ、以降では一般入力$(D,A,B,K)$ に対する理論整理と実装を示す。本稿の構成は次の通りである。
1. 問題設定（[問題設定](#問題設定)）では、誤差関数 $\Delta(D,A,B,x)$ と最小入力 $x_{\min}(D,A,B,K)$ を定義し、背景例を示す。
2. 解法（[解法](#解法)）では、式変形（[補題（不等式の同値変形）](#補題（不等式の同値変形）)）、解の存在条件と探索範囲（[補題（解の存在と探索上界）](#補題（解の存在と探索上界）)）、二分探索への帰着（[補題（最大安全長と最小解の一致）](#補題（最大安全長と最小解の一致）)）を順に示す。
3. `mwf` の詳細（[mwf の処理の詳細](#mwf-の処理の詳細)）では、分岐再帰（[補題（mwf の分岐再帰）](#補題（mwf-の分岐再帰）)）と Euclid 型反復に基づく評価法を整理する。
4. 計算量評価では、`mwf` の計算量評価（[mwf の計算量](#mwf-の計算量)）と`divapprox2` の計算量評価（[x_min の計算量評価](#x_min-の計算量評価)）を与える。
5. 実装節（[実装](#実装)）では、`mwf`（[mwf](#mwf)）、`mwf` の `floor_prod` 版（[floor_prod に基づく mwf の実装](#floorprod-に基づく-mwf-の実装)）、 `mwf_leq`, `divapprox2`（[divapprox2](#divapprox2)）の実装例を示す。

### 記号の説明

**実数の集合** $\mathbb{R}$ は実数全体の集合を表すものとする。

**自然数の集合** $\mathbb{N}$ は $\{0,1,2,\dots\}$ を表すものとする。

**整数の集合** $\mathbb{Z}$ は $\{\dots,-2,-1,0,1,2,\dots\}$ を表すものとする。

**非負整数の集合** $\mathbb{Z}_{\ge0}$ は $\{0,1,2,\dots\}$ を表すものとする。

**正整数の集合** $\mathbb{Z}_{>0}$ は $\{1,2,3,\dots\}$ を表すものとする。

**床関数** 実数 $x\in\mathbb{R}$ に対し、$n\le x < n+1$ を満たす唯一の整数 $n\in\mathbb{Z}$ を $\lfloor x\rfloor$ と書く。つまり、 $\lfloor x\rfloor$ は $x$ 以下の最大の整数である。

**天井関数** 実数 $x\in\mathbb{R}$ に対し、$n-1 < x \le n$ を満たす唯一の整数 $n\in\mathbb{Z}$ を $\lceil x\rceil$ と書く。つまり、 $\lceil x\rceil$ は $x$ 以上の最小の整数である。

**剰余** 整数 $x\in\mathbb{Z}$ と正整数 $m\in\mathbb{Z}_{>0}$ に対し、$x\bmod m$ はユークリッド除法の剰余（$0\le x\bmod m < m$）を表すものとする。

**整除** 整数 $a,b\in\mathbb{Z}$ に対し、$a\mid b$ は、ある整数 $k\in\mathbb{Z}$ が存在して $b=ak$ と書けることを表す。つまり、 $b$ は $a$ の倍数である。

**非整除** 整数 $a,b\in\mathbb{Z}$ に対し、$a\nmid b$ は、任意の整数 $k\in\mathbb{Z}$ に対して $b\ne ak$ となることを表す。つまり、 $b$ は $a$ の倍数ではない。

## 問題設定

$T$ 個のテストケースについてそれぞれ、正の整数 $D,A,B$ と非負整数 $K$ が与えられる。

#### 定義（誤差関数と最小閾値到達入力）

誤差関数を

$$
\Delta(D,A,B,x) := \left\lfloor\frac{x}{D}\right\rfloor-\left\lfloor\frac{\lfloor AB/D\rfloor\left\lfloor x/A\right\rfloor}{B}\right\rfloor
$$

さらに解が存在する場合に

$$
x_{\min}(D,A,B,K):=\min\left\{x\in\mathbb{Z}_{\ge0}\ \middle|\ K<\Delta(D,A,B,x)\right\}
$$

と定義する。

すなわち $x_{\min}(D,A,B,K)$ は、誤差 $\Delta(D,A,B,x)$ が閾値 $K$ を初めて上回る最小の非負整数入力 $x$ を表す。求めるのは、解が存在する場合はこの $x_{\min}(D,A,B,K)$、存在しない場合は $-1$ である。後述の制約の下では、出力が 64-bit 整数で表現可能な範囲を超える場合があるため、多倍長整数の利用が必要となる。

### 背景説明

本問題の背景として、区間 $0\le x<2^{128}$ における $\lfloor x/10^{19}\rfloor$ の高速計算を考える。直接除算を回避するため、$10^{19}$ による切り捨て除算の代替として「$2^{64}$ による切り捨て除算 $\to$ 定数倍 $\to$ $2^{64}$ による切り捨て除算」からなる近似を導入し、その誤差を評価する。

一般に、非負整数 $x$ に対し

$$
\Delta(D,A,B,x) := \left\lfloor\frac{x}{D}\right\rfloor-\left\lfloor\frac{\lfloor AB/D\rfloor\left\lfloor x/A\right\rfloor}{B}\right\rfloor
$$

を誤差関数と定義する。

ここで具体例として $D=10^{19},\ A=2^{64},\ B=2^{64}$ を代入すると

$$
\Delta(10^{19},2^{64},2^{64},x)=
\left\lfloor\frac{x}{10^{19}}\right\rfloor-
\left\lfloor
\frac{34028236692093846346 \left\lfloor x/2^{64}\right\rfloor}{2^{64}}
\right\rfloor
$$

となる。ここで

$$
\left\lfloor\frac{2^{128}}{10^{19}}\right\rfloor=34028236692093846346\simeq 1.84\times 2^{64}
$$

を用いた。

同パラメータでの $\Delta$ の評価として、以下が得られる。
- $0\le x<156624322790854844120000000000000000000\simeq 0.46\times 2^{128}$ では $0\le\Delta\le2$。
- $x=156624322790854844120000000000000000000\simeq 0.46\times 2^{128}$ では $\Delta=3$。
- $0\le x<1164985662416622730250000000000000000000\simeq 3.42\times 2^{128}$ では $0\le\Delta\le3$。
- $x=1164985662416622730250000000000000000000\simeq 3.42\times 2^{128}$ では $\Delta=4$。
- $2^{128}<1164985662416622730250000000000000000000\simeq 3.42\times 2^{128}$ より、$0\le x<2^{128}$ の範囲では $0\le\Delta\le3$ が保証される。

この観察に基づき、誤差 $\Delta$ が非負整数 $K$ を超える最小の $x$ を求める問題を考える。本稿ではこれを一般入力 $(D,A,B,K)$ に拡張して扱う。

なお、背景説明で用いた具体例 $D=10^{19},A=2^{64},B=2^{64}$ は、本問の入力制約（$D,A,B\le 10^{18}$）の範囲外である。

### 入出力と制約

**制限** 実行時間制限は 2.000 秒、メモリ制限は 512 MB とする。

**入力形式** 入力はプログラムの標準入力(`stdin`)から次の形式で与えられる。

```text
T
D_1 A_1 B_1 K_1
D_2 A_2 B_2 K_2
...
D_T A_T B_T K_T
```
ここで、$i+1$ 行目（$1\le i\le T$）は $i$ 番目のテストケース $(D_i,A_i,B_i,K_i)$ を表す。

**出力形式**

各テストケースにつき 1 行、解が存在する場合は$x_{\min}(D_i,A_i,B_i,K_i)$ を、存在しない場合は $-1$ をプログラムの標準出力(`stdout`)に出力する。

**制約**

- $1\le T\le 100$
- $1\le D,A,B\le 10^{18}$
- $0\le K\le 64$
- 入力はすべて整数

### サンプル（抜粋）

**入力**

```text
3
1 1 1 0
998244353 1000000000 1000000000 2
420196140727489673 679891637638612258 999999999999999989 7
```

**出力**

```text
-1
1391680038999995155
9007968880765895055838800035237338317
```

- $D=A=B=1,\ K=0$ のとき $\Delta(1,1,1,x)=0$ であり、条件を満たす $x$ は存在しない。
- $D=998244353,\ A=B=10^{9},\ K=2$ のとき、$\lfloor AB/D\rfloor=1001758734$ であり、$x=1391680038999995155$ にて $\Delta=3$ となる一方、$0\le x<1391680038999995155$ では $\Delta\le 2$ である。
- $D=420196140727489673,\ A=679891637638612258,\ B=999999999999999989,\ K=7$ のとき、出力すべき値が 64-bit 整数で表現できる範囲を超える。

## 解法

本節では、最小解の探索に向けた式変形と存在条件・探索範囲の評価、二分探索の枠組みを示す。

### 式変形

#### 補題（非倍数点での非増加）

任意の $x\in\mathbb{Z}_{\ge0}$ について、$D\nmid(x+1)$ ならば

$$
\Delta(D,A,B,x+1)\le \Delta(D,A,B,x)
$$

が成り立つ。特に、条件 $K<\Delta(D,A,B,x)$ を満たす最小の $x$ が存在するならば、それは必ず $D$ の倍数である。

**証明**

まず

$$
\Delta(D,A,B,x)=T_1(x)-T_2(x)
$$

とおき、

$$
T_1(x):=\left\lfloor\frac{x}{D}\right\rfloor,\qquad
T_2(x):=\left\lfloor\frac{\lfloor AB/D\rfloor\left\lfloor x/A\right\rfloor}{B}\right\rfloor
$$

と定義する。
このとき、第1項 $T_1$ は $D\nmid(x+1)$ のとき不変である一方、第2項 $T_2$ は $x$ の増加により非減少である。

実際、$\lfloor x/A\rfloor$ は $x$ に関して非減少であり、また $n_1\le n_2$ ならば

$$
\left\lfloor n_1\cdot\frac{\lfloor AB/D\rfloor}{B}\right\rfloor \le \left\lfloor n_2\cdot\frac{\lfloor AB/D\rfloor}{B}\right\rfloor
$$

が成り立つので、合成として第2項も $x$ に関して非減少である。例えば $D\nmid(x+1)$ のとき

$$
\left\lfloor\frac{x}{D}\right\rfloor=\left\lfloor\frac{x+1}{D}\right\rfloor
$$

が成り立つ。よって差 $\Delta$ は（同条件下で）増加し得ず、

$$
D\nmid(x+1)\ \Rightarrow\ \Delta(D,A,B,x+1)\le \Delta(D,A,B,x)
$$

が従う。したがって $\Delta$ が増える可能性があるのは $x\equiv -1\pmod{D}$ をまたぐ点に限られる。

ここで $t\equiv 0\pmod{D}$ とし、区間 $[t,t+D)$ を考える。もしこの区間内のある $x$ で $K<\Delta(D,A,B,x)$ が成立するならば、上の非増加性より左端 $t$ でも $K<\Delta(D,A,B,t)$ が成立する。ゆえに、条件 $K<\Delta(D,A,B,x)$ を満たす最小の $x$ が存在するならば、それは必ず $D$ の倍数に限られる。□

したがって $x=Du$ とおく。このとき

$$
u_{\min}(D,A,B,K):=\min\left\{u\in\mathbb{Z}_{\ge0}\ \middle|\ K<\Delta(D,A,B,Du)\right\}
$$

と定義すれば、解が存在する場合に $x_{\min}(D,A,B,K)=D\cdot u_{\min}(D,A,B,K)$ が成り立つ。

以降は

$$
M:=\left\lfloor \frac{AB}{D}\right\rfloor,\qquad R:=(AB)\bmod D=AB-DM
$$

とおく。このとき

$$
\begin{aligned}
\Delta(D,A,B,Du)
&= \left\lfloor\frac{Du}{D}\right\rfloor-\left\lfloor\left\lfloor\frac{Du}{A}\right\rfloor\cdot\frac{M}{B}\right\rfloor \\
&= u-\left\lfloor\frac{M\left\lfloor Du/A\right\rfloor}{B}\right\rfloor.
\end{aligned}
$$

#### 補題（不等式の同値変形）

$M:=\left\lfloor \frac{AB}{D}\right\rfloor$ および $R:=(AB)\bmod D$ とおくと、任意の $u\in\mathbb{Z}_{\ge 0}$ について次が同値である。ここで $(Du)\bmod A$ はユークリッド除法の剰余（$0\le (Du)\bmod A < A$）を表すものとする：

$$
K<\Delta(D,A,B,Du)
\iff BK< Bu-M\left\lfloor\frac{Du}{A}\right\rfloor
\iff ABK< \bigl((Du)\bmod A\bigr)M + Ru.
$$

**証明**

以下で逐次変形する。

$$
\begin{aligned}
K<\Delta(D,A,B,Du)
&\iff K+1\le \Delta(D,A,B,Du)
&& (K,\Delta\in\mathbb{Z}\ \Rightarrow\ K<\Delta\iff K+1\le \Delta) \\
&\iff K+1\le u-\left\lfloor \left\lfloor\frac{Du}{A}\right\rfloor\cdot\frac{M}{B}\right\rfloor
&& (\Delta(D,A,B,Du)=u-\lfloor\lfloor Du/A\rfloor\cdot M/B\rfloor) \\
&\iff \left\lfloor \left\lfloor\frac{Du}{A}\right\rfloor\cdot\frac{M}{B}\right\rfloor\le u-K-1 \\
&\iff \left\lfloor\frac{Du}{A}\right\rfloor\cdot\frac{M}{B}< u-K
&& (\lfloor r\rfloor\le z\iff r<z+1) \\
&\iff M\left\lfloor\frac{Du}{A}\right\rfloor < B(u-K)
&& (B>0\ \Rightarrow\ \text{両辺に }B\text{ を乗じる}) \\
&\iff BK< Bu-M\left\lfloor\frac{Du}{A}\right\rfloor.
\end{aligned}
$$

さらに

$$
\left\lfloor\frac{Du}{A}\right\rfloor=\frac{Du-((Du)\bmod A)}{A}
$$

および $AB=DM+R$ を用いると

$$
\begin{aligned}
A\bigl(Bu-M\lfloor Du/A\rfloor\bigr)
&=ABu-AM\cdot\frac{Du-((Du)\bmod A)}{A} \\
&=ABu-DMu+M((Du)\bmod A) \\
&=(AB-DM)u+M((Du)\bmod A) \\
&=Ru+M((Du)\bmod A)
\end{aligned}
$$

であるから、両辺に正の整数 $A$ を掛けることで

$$
BK< Bu-M\left\lfloor\frac{Du}{A}\right\rfloor
\iff ABK< \bigl((Du)\bmod A\bigr)M + Ru
$$

が従う。□

### 解の存在条件と探索範囲

#### 補題（剰余値域の gcd 制約（上界付き））

正の整数 $D,A$ を仮定し、$g:=\gcd(D,A)$ とする。任意の $u\in\mathbb{Z}_{\ge0}$ について

$$
(Du)\bmod A \equiv 0 \pmod g
$$

が成り立つ。したがってある整数 $t$（$0\le t< A/g$）が存在して

$$
(Du)\bmod A = g\,t
$$

と表せる。特に

$$
0\le (Du)\bmod A \le A-g
$$

が成り立つ。

**証明**

ユークリッド除法により

$$
Du = A q + r,\qquad 0\le r < A
$$

と一意に書け、この $r$ が $(Du)\bmod A$ である。$g\mid D$ より $g\mid Du$、また $g\mid A$ より $g\mid Aq$ なので差を取って $g\mid r$。よって $r=g t$ と書ける。

さらに $0\le r<A=g\cdot(A/g)$ より $0\le t< A/g$ が従う。最後に $t$ は整数なので $t\le A/g-1$、したがって

$$
r=gt \le g\left(\frac{A}{g}-1\right)=A-g
$$

が成り立つ。□

#### 補題（Bézout による最大剰余 A-g の達成）

正の整数 $D,A$ を仮定し、 $g:=\gcd(D,A)$ とし $D':=D/g,\ A':=A/g$ とおく（このとき $\gcd(D',A')=1$）。するとある $u_*\in\mathbb{Z}_{\ge0}$（実際 $0\le u_*<A'$）が存在して

$$
(Du_*)\bmod A = A-g
$$

が成り立つ。

**証明**

$\gcd(D',A')=1$ なので Bézout の等式により、ある整数 $p,q\in\mathbb{Z}$ が存在して

$$
D' p + A' q = 1
$$

が成り立つ。両辺を法 $A'$ で見ると

$$
D' p \equiv 1 \pmod{A'}
$$

である。ここで $u_*:=(-p)\bmod A'$（ユークリッド除法の剰余、すなわち $0\le u_*<A'$ かつ $u_*\equiv -p\pmod{A'}$）とおくと

$$
D' u_* \equiv -D' p \equiv -1 \equiv A'-1 \pmod{A'}
$$

が成り立つ。両辺に $g$ を掛けて法 $A=gA'$ で見れば

$$
D u_* \equiv g(A'-1)=A-g \pmod A
$$

である。一方で $(Du_*)\bmod A$ は $[0,A)$ の唯一の代表元なので、上の合同式から

$$
(Du_*)\bmod A = A-g
$$

が従う。□

#### 補題（解の存在条件（R が 0 でない場合））

$R\neq0$ ならば、条件 $K<\Delta(D,A,B,Du)$ を満たす $u\in\mathbb{Z}_{\ge0}$ が存在する。

**証明**

[補題（不等式の同値変形）](#補題（不等式の同値変形）)より、示すべきことは

$$
ABK<\bigl((Du)\bmod A\bigr)M+Ru
$$

を満たす $u$ の存在である。$R\neq0$ より $R\ge1$ である。ここで $u:=ABK+1$ とおくと $u\ge0$ なので

$$
u\le Ru
$$

が成り立つ。また $(Du)\bmod A\ge0,\ M\ge0$ なので

$$
Ru\le \bigl((Du)\bmod A\bigr)M+Ru
$$

である。したがって

$$
ABK<u\le Ru\le \bigl((Du)\bmod A\bigr)M+Ru
$$

が成り立ち、結論が従う。□

#### 補題（解の存在条件（R が 0 の場合））

$R=0$ のとき、条件 $K<\Delta(D,A,B,Du)$ を満たす $u$ が存在することは

$$
DK+g<A
$$

と同値である。

**証明**

[補題（不等式の同値変形）](#補題（不等式の同値変形）)より、$R=0$ の下で

$$
K<\Delta(D,A,B,Du)\iff ABK<\bigl((Du)\bmod A\bigr)M
$$

である。[補題（剰余値域の gcd 制約（上界付き））](#補題（剰余値域の-gcd-制約（上界付き））)から $((Du)\bmod A)\le A-g$ なので、解が存在すれば

$$
ABK<\bigl((Du)\bmod A\bigr)M\le (A-g)M
$$

が必要である。逆に $ABK<(A-g)M$ なら、[補題（Bézout による最大剰余 A-g の達成）](#補題（bézout-による最大剰余-a-g-の達成）)により$((Du)\bmod A)=A-g$ を満たす $u$ があり、同不等式を満たす。

したがって存在条件は $ABK<(A-g)M$ と同値である。ここで $R=0$ なら$AB=DM$ かつ $M\ge1$ なので

$$
ABK<(A-g)M\iff DMK<(A-g)M\iff DK<A-g
$$

であり、整数条件として

$$
DK<A-g\iff DK+g<A
$$

を得る。□

#### 補題（最小解の下界）

解が存在するとき

$$
K+1\le u_{\min}(D,A,B,K)
$$

が成り立つ。

**証明**

任意の $u$ について

$$
\Delta(D,A,B,Du)=u-\left\lfloor\dfrac{M\left\lfloor Du/A\right\rfloor}{B}\right\rfloor\le u
$$

である。ゆえに $K<\Delta(D,A,B,Du)$ なら $K+1\le u$ が必要であり、最小解にも適用できる。□

#### 補題（探索上界（R が 0 でない場合））

$R\neq0$ かつ解が存在するとき

$$
\max\left(\left\lfloor\frac{ABK-(A-g)M}{R}\right\rfloor+1,\ K+1\right)
\le u_{\min}(D,A,B,K)
<\left\lfloor\frac{ABK}{R}\right\rfloor+2
$$

が成り立つ。

**証明**

[補題（剰余値域の gcd 制約（上界付き））](#補題（剰余値域の-gcd-制約（上界付き））)より $((Du)\bmod A)\le A-g$ なので

$$
ABK<\bigl((Du)\bmod A\bigr)M+Ru\le (A-g)M+Ru
$$

から

$$
u>\frac{ABK-(A-g)M}{R}
$$

が従う。よって整数性により

$$
u\ge \left\lfloor\frac{ABK-(A-g)M}{R}\right\rfloor+1.
$$

また[補題（最小解の下界）](#補題（最小解の下界）)から $u\ge K+1$。以上を最小解に適用して左側を得る。

右側は $u_0:=\left\lfloor\dfrac{ABK}{R}\right\rfloor+1$ とおけば $Ru_0>ABK$ なので

$$
ABK<Ru_0\le \bigl((Du_0)\bmod A\bigr)M+Ru_0
$$

となる。[補題（不等式の同値変形）](#補題（不等式の同値変形）)より $u_0$ は条件を満たし、したがって

$$
u_{\min}\le u_0<\left\lfloor\frac{ABK}{R}\right\rfloor+2.
$$

□

#### 補題（探索上界（R が 0 の場合））

$R=0$ かつ解が存在するとき

$$
u_{\min}(D,A,B,K)<\frac{A}{g}
$$

が成り立つ。

**証明**

[補題（解の存在条件（$R$ が 0 の場合））](#補題（解の存在条件（r-が-0-の場合））)から $DK+g<A$、すなわち $DK\le A-g-1$。したがって

$$
DMK\le (A-g-1)M<(A-g)M
$$

であり、$R=0$ による $AB=DM$ から

$$
ABK<(A-g)M
$$

を得る。[補題（Bézout による最大剰余 A-g の達成）](#補題（bézout-による最大剰余-a-g-の達成）)から $0\le u_*<A/g$ かつ$((Du_*)\bmod A)=A-g$ を満たす $u_*$ が存在するので

$$
ABK<\bigl((Du_*)\bmod A\bigr)M+Ru_*=(A-g)M
$$

である。[補題（不等式の同値変形）](#補題（不等式の同値変形）)から $u_*$ は条件を満たし、最小性より$u_{\min}\le u_*<A/g$。□

#### 補題（共通上界）

解が存在するなら

$$
u_{\min}(D,A,B,K)<ABK+2
$$

が成り立つ。

**証明**

$R\neq0$ のときは[補題（探索上界（$R$ が 0 でない場合））](#補題（探索上界（r-が-0-でない場合））)と $R\ge1$ より

$$
u_{\min}\le \left\lfloor\frac{ABK}{R}\right\rfloor+1\le ABK+1<ABK+2.
$$

$R=0$ のときは場合分けする。$K=0$ なら[補題（解の存在条件（$R$ が 0 の場合））](#補題（解の存在条件（r-が-0-の場合））)から $g<A$。ゆえに $A\nmid D$ であり $(D\bmod A)\ge1$。また $M\ge1$ なので

$$
0=ABK<(D\bmod A)M=\bigl((D\cdot1)\bmod A\bigr)M
$$

となり、$u=1$ が条件を満たす。したがって $u_{\min}\le1=ABK+1$。

$K>0$ なら[補題（探索上界（$R$ が 0 の場合））](#補題（探索上界（r-が-0-の場合））)から $u_{\min}<A/g$。さらに $K\ge1,\ B\ge1$ より $ABK\ge A\ge A/g$ であるから

$$
u_{\min}<A/g\le ABK\le ABK+1.
$$

以上より常に $u_{\min}\le ABK+1<ABK+2$。□

#### 補題（解の存在と探索上界）

$g:=\gcd(D,A)$ とする。
- $R\neq 0$ ならば、条件 $K<\Delta(D,A,B,Du)$ を満たす $u$ は必ず存在する。
- $R=0$ ならば、条件 $K<\Delta(D,A,B,Du)$ を満たす $u$ が存在することは $DK+g<A$ と同値である。

また、解が存在する場合、最小解 $u_{\min}(D,A,B,K)$ には有限な探索上界が存在し、例えば

$$
R=0\Rightarrow u_{\min}<\frac{A}{g},\qquad
R\neq0\Rightarrow u_{\min}<\left\lfloor\frac{ABK}{R}\right\rfloor+2
$$

が成り立つ。さらに、どちらの場合にも共通の粗い上界として

$$
u_{\min}(D,A,B,K)<ABK+2
$$

が成り立つ。

**証明**

存在条件は[補題（解の存在条件（$R$ が 0 でない場合））](#補題（解の存在条件（r-が-0-でない場合））), [補題（解の存在条件（$R$ が 0 の場合））](#補題（解の存在条件（r-が-0-の場合））)、場合別上界は[補題（探索上界（$R$ が 0 の場合））](#補題（探索上界（r-が-0-の場合））), [補題（探索上界（$R$ が 0 でない場合））](#補題（探索上界（r-が-0-でない場合））)、共通上界は[補題（共通上界）](#補題（共通上界）)による。□

### 二分探索による最小解の決定

本節では、条件 $K<\Delta(D,A,B,Du)$ を満たす最小の $u$ を求めるために、二分探索を用いる。

まず[補題（不等式の同値変形）](#補題（不等式の同値変形）)より、条件 $K<\Delta(D,A,B,Du)$ は

$$
BK< Bu-M\left\lfloor\frac{Du}{A}\right\rfloor
$$

と同値である。したがって $N$ を固定したとき、区間 $u\in\{0,1,\dots,N-1\}$ の中にこの不等式を満たす $u$ が存在するかどうかは、右辺 $Bu-M\lfloor Du/A\rfloor$ の最大値が $BK$ を超えるかどうかに帰着される。

この発想に基づき、まず

$$
f(u):=Bu-M\left\lfloor\frac{Du}{A}\right\rfloor
$$

とおく。

さらに任意の整数 $L<R$ に対して

$$
F(L,R):=\max_{L\le u<R}\left(Bu-M\left\lfloor\frac{Du}{A}\right\rfloor\right)
$$

と定義する。

定義より $F(0,N)=\max_{0\le u<N} f(u)$ である。
#### 補題（prefix hit 判定）

任意の整数 $N>0$ について

$$
\exists u\in\{0,1,\dots,N-1\}\ \bigl(K<\Delta(D,A,B,Du)\bigr)
\iff F(0,N)>BK
$$

が成り立つ。

**証明**

[補題（不等式の同値変形）](#補題（不等式の同値変形）)より$K<\Delta(D,A,B,Du)\iff BK<f(u)$ である。したがって左辺は

$$
\exists u\in\{0,1,\dots,N-1\}\ (f(u)>BK)
$$

と同値であり、これは $F(0,N)=\max_{0\le u<N}f(u)$ の定義から右辺と同値である。□

#### 補題（最大安全長の表式）

解が存在すると仮定し、

$$
u_0:=\min\left\{u\in\mathbb{Z}_{\ge0}\ \middle|\ K<\Delta(D,A,B,Du)\right\}
$$

とおく。このとき

$$
u_0=\max\left\{N\in\mathbb{Z}_{>0}\ \middle|\ F(0,N)\le BK\right\}
$$

が成り立つ。

**証明**

[補題（prefix hit 判定）](#補題（prefix-hit-判定）)より

$$
F(0,N)\le BK
\iff
\nexists u\in\{0,1,\dots,N-1\}\ \bigl(K<\Delta(D,A,B,Du)\bigr).
$$

$u_0$ の最小性を使えば、右辺はちょうど $N\le u_0$ と同値である。よって$F(0,N)\le BK$ を満たす正整数 $N$ の最大値は $u_0$。□

#### 補題（区間分割と単調性）

任意の整数 $0<n<N$ について

$$
F(0,N)=\max\bigl(F(0,n),F(n,N)\bigr)
$$

が成り立つ。特に

$$
F(0,n)\le F(0,N)
$$

である。

**証明**

区間 $[0,N)$ の最大値を $[0,n)$ と $[n,N)$ の最大値に分割しただけである。□

#### 補題（後半区間への判定移送）

整数 $0<n<N$ で $F(0,n)\le BK$ を仮定する。このとき

$$
F(0,N)\le BK \iff F(n,N)\le BK
$$

が成り立つ。

**証明**

[補題（区間分割と単調性）](#補題（区間分割と単調性）)の

$$
F(0,N)=\max(F(0,n),F(n,N))
$$

を用いる。仮定 $F(0,n)\le BK$ の下では、

$$
\max(F(0,n),F(n,N))\le BK \iff F(n,N)\le BK
$$

である。□

以上より、$F(0,N)\le BK$ が成り立つ最大の $N$ を二分探索で求めればよい。この最大値は[補題（最大安全長の表式）](#補題（最大安全長の表式）)により $u_{\min}(D,A,B,K)$ に一致し、したがって $x_{\min}(D,A,B,K)=D\cdot u_{\min}(D,A,B,K)$ である。

また、区間 $[l,r)$ 上での関数 $x\mapsto ax+b\left\lfloor\frac{cx+d}{m}\right\rfloor$ の最大値を

$$
\mathtt{mwf}(l,r,m,a,b,c,d):=\max\left\{ax+b\left\lfloor\frac{cx+d}{m}\right\rfloor\ \middle|\ l\le x<r\right\}
$$

と定義すれば、

$$
F(L,R)=\mathtt{mwf}(L,R,A,B,-M,D,0)
$$

が成り立つ。

#### 補題（最大安全長と最小解の一致）

解が存在する場合、[補題（解の存在と探索上界）](#補題（解の存在と探索上界）)により有限な上界 $U$ が取れるので、

$$
N_{\max}:=\max\left\{N\in\mathbb{Z}_{>0}\ \middle|\ F(0,N)\le BK\right\}
$$

を二分探索で求めればよい。このとき $u_{\min}(D,A,B,K)=N_{\max}$ であり、したがって $x_{\min}(D,A,B,K)=D\,N_{\max}$ である。

**証明**

解が存在するので、$u_{\min}(D,A,B,K)$ は

$$
u_0:=\min\{u\in\mathbb{Z}_{\ge0}\mid K<\Delta(D,A,B,Du)\}
$$

で与えられる。[補題（最大安全長の表式）](#補題（最大安全長の表式）)をこの $u_0$ に適用すると

$$
u_{\min}(D,A,B,K)
=\max\{N\in\mathbb{Z}_{>0}\mid F(0,N)\le BK\}
=N_{\max}
$$

を得る。

また[補題（区間分割と単調性）](#補題（区間分割と単調性）)により述語 $P(N):\iff F(0,N)\le BK$ は$N$ に関して単調であり、[補題（解の存在と探索上界）](#補題（解の存在と探索上界）)から有限上界 $U$ が取れるので、$N_{\max}$ は二分探索で求められる。

さらに $Bu-M\lfloor Du/A\rfloor$ は $ax+b\lfloor(cx+d)/m\rfloor$ 型（$a=B,b=-M,c=D,m=A,d=0$）であるから

$$
F(0,N)=\mathtt{mwf}(0,N,A,B,-M,D,0)
$$

として `mwf` で評価できる。

**区間判定**

二分探索では、常に区間 $u\in[0,\mathtt{lo})$ に条件 $BK<Bu-M\lfloor Du/A\rfloor$ を満たす $u$ が存在しない（すなわち $F(0,\mathtt{lo})\le BK$）という不変条件を保つ。その上で、区間 $u\in[\mathtt{lo},\mathtt{mid})$ に条件 $BK<Bu-M\lfloor Du/A\rfloor$ を満たす $u$ が存在しないことを判定したい。これは[補題（後半区間への判定移送）](#補題（後半区間への判定移送）)により

$$
F(\mathtt{lo},\mathtt{mid})\le BK
$$

の判定に帰着し、すなわち

$$
\max_{\mathtt{lo}\le u<\mathtt{mid}}\left(Bu-M\left\lfloor\frac{Du}{A}\right\rfloor\right)\le BK
$$

と同値である。□

#### 補題（`x_min` が X 未満であることの判定の帰着）

解が存在し、$N:=\left\lceil \frac{X}{D}\right\rceil>0$ とする。このとき

$$
x_{\min}(D,A,B,K)<X\iff F(0,N)>BK
$$

が成り立つ。

**証明**

$N=\left\lceil X/D\right\rceil$ は $X\le D\cdot u$ を満たす最小整数なので、$x_{\min}=D\cdot u_{\min}$ より

$$
x_{\min}<X\iff u_{\min}<N
$$

である。さらに $u_{\min}$ の最小性から

$$
u_{\min}<N
\iff
\exists u\in\{0,1,\dots,N-1\}\ \bigl(K<\Delta(D,A,B,Du)\bigr).
$$

右辺は[補題（prefix hit 判定）](#補題（prefix-hit-判定）)により $F(0,N)>BK$ と同値である。□

したがって、この判定は $F(0,N)=\mathtt{mwf}(0,N,A,B,-M,D,0)$（あるいは閾値判定版）を 1 回評価するだけで行える。

**出力への戻し方**

上の二分探索は $u_{\min}(D,A,B,K)$ を返すので、求める答えは

$$
x_{\min}(D,A,B,K)=D\cdot u_{\min}(D,A,B,K)
$$

である。

### `mwf` の処理の詳細

以降では、先に定義した $\mathtt{mwf}$ を効率的に計算する手続きを述べる。

$$
\mathtt{mwf}(l,r,m,a,b,c,d):=\max_{l\le x<r}\left(ax+b\left\lfloor\frac{cx+d}{m}\right\rfloor\right)
$$

ここで $l,r,m,a,b,c,d$ はすべて整数であり、$l<r$ および $m>0$ とする。

この値を、非再帰ループ（もしくは末尾再帰）により $\mathcal{O}(\log m)$ 程度の反復で計算する。以降の各局面では、小問題を $\mathtt{mwf}(0,n,m,a,b,c,d)$ の形に保ちつつ、補助変数 $\mathtt{maxAcc},\mathtt{sumAcc}$ を更新して

$$
\max\bigl(\mathtt{maxAcc},\ \mathtt{sumAcc}+\mathtt{mwf}(0,n,m,a,b,c,d)\bigr)
$$

を不変に保つ、という視点で議論する。すなわち、平行移動・正規化・端点評価・縮約の各操作は、いずれもこの不変量を保存する更新則として統一的に読める。

以下の小節では、再帰式を平行移動・正規化・端点評価・$a$ の符号による場合分けの順に導出し、最後に分岐再帰としてまとめる。

#### 原問題の平行移動

#### 補題（平行移動による基準化）

$n:=r-l$ とする。このとき

$$
\mathtt{mwf}(l,r,m,a,b,c,d)
=al+b\left\lfloor\frac{cl+d}{m}\right\rfloor+\mathtt{mwf}(0,n,m,a,b,c,(cl+d)\bmod m)
$$

が成り立つ。

**証明**

$q:=\left\lfloor\dfrac{cl+d}{m}\right\rfloor,\ d':=(cl+d)\bmod m$ とおく。任意の $x\in\mathbb{Z}$ に対し $x=l+t$（$0\le t<n$）と書くと

$$
\begin{aligned}
a x+b\left\lfloor\frac{cx+d}{m}\right\rfloor
&=a(l+t)+b\left\lfloor\frac{c(l+t)+d}{m}\right\rfloor \\
&=al+at+b\left\lfloor\frac{cl+d+ct}{m}\right\rfloor \\
&=al+at+b\left(q+\left\lfloor\frac{ct+d'}{m}\right\rfloor\right) \\
&=al+bq+\left(at+b\left\lfloor\frac{ct+d'}{m}\right\rfloor\right).
\end{aligned}
$$

したがって $t$ について最大を取れば

$$
\begin{aligned}
\mathtt{mwf}(l,r,m,a,b,c,d)
&=al+bq+\max_{0\le t<n}\left(at+b\left\lfloor\frac{ct+d'}{m}\right\rfloor\right) \\
&=al+b\left\lfloor\frac{cl+d}{m}\right\rfloor+\mathtt{mwf}(0,n,m,a,b,c,(cl+d)\bmod m)
\end{aligned}
$$

を得る。□

**実装上の補足**

実装では端点候補を逐次確定させるための管理変数として$\mathtt{maxAcc}$（これまでに確定した端点最大）と $\mathtt{sumAcc}$（定数項累積）を用いる。平行移動直後の初期値は

$$
\mathtt{maxAcc}=\mathtt{sumAcc}=a l + b\left\lfloor\dfrac{cl+d}{m}\right\rfloor
$$

であり、さらに $d':=(cl+d)\bmod m$ とすると

$$
\mathtt{mwf}(l,r,m,a,b,c,d)
=\max\bigl(\mathtt{maxAcc},\ \mathtt{sumAcc}+\mathtt{mwf}(0,n,m,a,b,c,d')\bigr)
$$

という不変量を保ちながら小問題を更新できる。Python 実装中の対応する変数名は `max_acc`, `sum_acc` である。

#### 正規化（c,d が 0 以上 m 未満）

#### 補題（係数の正規化）

任意の整数 $x$ について

$$
ax+b\left\lfloor\frac{cx+d}{m}\right\rfloor
=b\left\lfloor\frac{d}{m}\right\rfloor+\left(a+b\left\lfloor\dfrac{c}{m}\right\rfloor\right)x+b\left\lfloor\frac{(c\bmod m)x+(d\bmod m)}{m}\right\rfloor
$$

が成り立つ。

**証明**

商と剰余を

$$
q_c:=\left\lfloor\frac{c}{m}\right\rfloor,\quad c':=c\bmod m,\qquad
q_d:=\left\lfloor\frac{d}{m}\right\rfloor,\quad d':=d\bmod m
$$

とおくと、$m>0$ より

$$
c=mq_c+c',\ 0\le c'<m,\qquad d=mq_d+d',\ 0\le d'<m
$$

である。したがって

$$
\begin{aligned}
\left\lfloor\frac{cx+d}{m}\right\rfloor
&=\left\lfloor\frac{(mq_c+c')x+(mq_d+d')}{m}\right\rfloor \\
&=\left\lfloor q_cx+q_d+\frac{c'x+d'}{m}\right\rfloor \\
&=q_cx+q_d+\left\lfloor\frac{c'x+d'}{m}\right\rfloor
\end{aligned}
$$

を得る（$q_cx+q_d$ は整数）。よって両辺に $b$ を掛けて $ax$ を加えると

$$
\begin{aligned}
ax+b\left\lfloor\frac{cx+d}{m}\right\rfloor
&=ax+b\left(q_cx+q_d+\left\lfloor\frac{c'x+d'}{m}\right\rfloor\right) \\
&=bq_d+(a+bq_c)x+b\left\lfloor\frac{c'x+d'}{m}\right\rfloor
\end{aligned}
$$

となる。最後に$q_c=\left\lfloor\tfrac{c}{m}\right\rfloor,\ q_d=\left\lfloor\tfrac{d}{m}\right\rfloor,\ c'=c\bmod m,\ d'=d\bmod m$を戻せば主張が従う。□

本補題の利用として、後段の階段関数の議論では$c\leftarrow c\bmod m,\ d\leftarrow d\bmod m$ に正規化する。その際、定数項 $b\lfloor d/m\rfloor$ を $\mathtt{sumAcc}$ に加算し、$a\leftarrow a+b\lfloor c/m\rfloor$ と更新すればよい。

#### 端点更新

#### 補題（端点での階段関数最大値）

正規化後に $0\le c,d<m$、$n>0$ とする。

$$
Y(x):=\left\lfloor\frac{cx+d}{m}\right\rfloor\qquad(x\in\mathbb{Z}_{\ge0})
$$

とおくと、次が成り立つ。
1. （単調性）$Y$ は $\mathbb{Z}_{\ge0}$ 上で単調非減少である。
2. （区間最大）$\displaystyle \mathtt{yMax}:=\max_{0\le x<n}Y(x)$ とおくと

$$
\mathtt{yMax}=Y(n-1)=\left\lfloor\frac{c(n-1)+d}{m}\right\rfloor
$$

が成り立つ。

**証明**

**(1)**
任意の $x\in\mathbb{Z}_{\ge0}$ について $c\ge0$ より $cx+d\le c(x+1)+d$ である。

$f(t):=\left\lfloor \dfrac{t}{m}\right\rfloor$ は $t$ に関して単調非減少なので、

$$
Y(x)=f(cx+d)\le f(c(x+1)+d)=Y(x+1)
$$

が従い、$Y$ は単調非減少である。

**(2)**
$Y$ が単調非減少なので、区間 $\{0,1,\dots,n-1\}$ における最大値は右端 $x=n-1$ で達成される。

よって

$$
\max_{0\le x<n}Y(x)=Y(n-1)=\left\lfloor\frac{c(n-1)+d}{m}\right\rfloor
$$

が成り立つ。□

Python 実装中の対応する変数名は `y_max` である。まず端点 $x=0$ および $x=n-1$ の値は常に最大候補として評価し、

$$
\phi(x):=ax+b\lfloor(cx+d)/m\rfloor
$$

と定義して、

$$
\mathtt{maxAcc}\leftarrow\max\bigl(\mathtt{maxAcc},\ \mathtt{sumAcc},\ \mathtt{sumAcc}+\phi(n-1)\bigr)
$$

のように更新する。なお正規化後は $0\le d<m$ なので $\phi(0)=b\left\lfloor\dfrac{d}{m}\right\rfloor=0$ が成り立つ（したがって $\mathtt{sumAcc}=\mathtt{sumAcc}+\phi(0)$ とみなせる）。

#### 終了条件

#### 補題（端点で終了できる条件）

ここまでの処理で得られた小問題

$$
\mathtt{mwf}(0,n,m,a,b,c,d)=\max_{x\in\mathbb{Z}_{\ge0},\ 0\le x<n}\left(a x + b\left\lfloor\frac{c x + d}{m}\right\rfloor\right)
$$

（$a,c,d$ は正規化後）に対し、次のいずれかが成り立てば最大値は端点で確定する：
- $a\le0$ かつ $b\le0$。
- $a\ge0$ かつ $b\ge0$。
- $\mathtt{yMax}=0$。

**証明**

- $a\le0$ かつ $b\le0$ なら、$x$ 増加に伴い $ax$ と $bY(x)$ がともに非増加で、最大は $x=0$。
- $a\ge0$ かつ $b\ge0$ なら、同様に非減少で、最大は $x=n-1$。
- $\mathtt{yMax}=0$ なら $Y(x)\equiv 0$ で $\phi(x)=ax$。よって $a\ge0$ なら $x=n-1$、$a\le0$ なら $x=0$ で最大となる。

□

#### 区間端点の式

#### 補題（階段区間の端点公式）

正規化後に $0\le c,d<m$、$n>0$ とする。

$$
Y(x):=\left\lfloor\frac{cx+d}{m}\right\rfloor\qquad(x\in\mathbb{Z}_{\ge0})
$$

とおき、

$$
\mathtt{yMax}:=Y(n-1)=\left\lfloor\frac{c(n-1)+d}{m}\right\rfloor
$$

とする。さらに $\mathtt{yMax}\ge1$ を仮定する（このとき $0<c<m$ が成り立つ）。

各 $y\in\{0,1,\dots,\mathtt{yMax}\}$ について

$$
I(y):=\{x\in\mathbb{Z}_{\ge0}\mid 0\le x<n,\ Y(x)=y\}
$$

とおく。このとき次が成り立つ。
1. （単調性と増分）
任意の $x\in\mathbb{Z}_{\ge0}$ について

$$
Y(x)\le Y(x+1)\le Y(x)+1,
\qquad\text{すなわち}\qquad
Y(x+1)-Y(x)\in\{0,1\}
$$

が成り立つ。特に $Y$ は単調非減少であり、また $Y(0)=0$ かつ $Y(n-1)=\mathtt{yMax}$ である。

2. （$y<\mathtt{yMax}$ の右端公式）
任意の $y$（$0\le y<\mathtt{yMax}$）について

$$
r(y):=\left\lceil\frac{m(y+1)-d}{c}\right\rceil-1
=\left\lfloor\frac{m(y+1)-d-1}{c}\right\rfloor
$$

とおくと、$r(y)=\max I(y)$ が成り立つ（したがって $r(y)\in I(y)$）。

3. （最終段 $y=\mathtt{yMax}$）

$$
r(\mathtt{yMax}):=n-1
$$

とおけば $r(\mathtt{yMax})=\max I(\mathtt{yMax})$ が成り立つ。

4. （左端の閉形式と隣接関係）

$$
\ell(0):=0,\qquad
\ell(y):=\left\lceil\frac{my-d}{c}\right\rceil
:=\left\lfloor\frac{my-d+c-1}{c}\right\rfloor\quad(1\le y\le \mathtt{yMax})
$$

とおくと、$\ell(y)=\min I(y)$ が成り立つ。さらに $1\le y\le \mathtt{yMax}$ について

$$
\ell(y)=r(y-1)+1\qquad(1\le y\le \mathtt{yMax})
$$

が成り立つ。

**証明**

$\mathtt{yMax}\ge1$ より $c(n-1)+d\ge m$ である。$0\le d<m$ なので $c(n-1)\ge1$、
よって $n\ge2$ かつ $c\ge1$ である。正規化より $c<m$ なので $0<c<m$。

**(1)**
$0<c<m$ より

$$
Y(x+1)=\left\lfloor\frac{c(x+1)+d}{m}\right\rfloor
=\left\lfloor\frac{cx+d}{m}+\frac{c}{m}\right\rfloor
$$

である。ここで $0<\frac{c}{m}<1$ なので一般に
$\lfloor r\rfloor \le \lfloor r+\alpha\rfloor \le \lfloor r\rfloor+1\ (0\le\alpha<1)$
が成り立つことから

$$
Y(x)\le Y(x+1)\le Y(x)+1
$$

を得る。従って $Y$ は単調非減少である。また $0\le d<m$ より $Y(0)=0$、定義より $Y(n-1)=\mathtt{yMax}$。

**(2)**
$0\le y<\mathtt{yMax}$ とし

$$
r:=\left\lfloor\frac{m(y+1)-d-1}{c}\right\rfloor
$$

とおく。まず $0\le d<m$ より $m(y+1)-d-1\ge m-d-1\ge0$ なので $r\ge0$、従って $r\in\mathbb{Z}_{\ge0}$。

床関数の定義から

$$
c r \le m(y+1)-d-1 < c(r+1)
$$

である。両辺に $d$ を足すと

$$
c r + d \le m(y+1)-1,\qquad c(r+1)+d \ge m(y+1)
$$

を得る。よって

$$
Y(r)=\left\lfloor\frac{cr+d}{m}\right\rfloor \le y,\qquad
Y(r+1)=\left\lfloor\frac{c(r+1)+d}{m}\right\rfloor \ge y+1
$$

が成り立つ。

次に $r\ge n-1$ と仮定すると単調性より

$$
\mathtt{yMax}=Y(n-1)\le Y(r)\le y
$$

となり $y<\mathtt{yMax}$ に矛盾する。従って $r\le n-2$、よって $r+1\le n-1$ であり $0\le r<n$ が成り立つ。

上で $Y(r)\le y$ と $Y(r+1)\ge y+1$ を得ており、さらに (1) より
$Y(r+1)\le Y(r)+1$ なので

$$
y+1\le Y(r+1)\le Y(r)+1\le y+1
$$

が従い、結局 $Y(r+1)=y+1$ かつ $Y(r)=y$ である。よって $r\in I(y)$。

さらに任意の $x>r$ については $x\ge r+1$ より単調性から
$Y(x)\ge Y(r+1)=y+1$、従って $x\notin I(y)$。
ゆえに $x\in I(y)\Rightarrow x\le r$、すなわち $r=\max I(y)$。

最後に（$c>0$ より）一般に

$$
\left\lceil\frac{u}{c}\right\rceil-1=\left\lfloor\frac{u-1}{c}\right\rfloor
\qquad(u\in\mathbb{Z})
$$

が成り立つので、$u=m(y+1)-d$ を代入すれば

$$
\left\lceil\frac{m(y+1)-d}{c}\right\rceil-1
=\left\lfloor\frac{m(y+1)-d-1}{c}\right\rfloor
$$

を得る。

**(3)**
$r(\mathtt{yMax})=n-1$ と定めれば $Y(n-1)=\mathtt{yMax}$ より $n-1\in I(\mathtt{yMax})$。
また $I(\mathtt{yMax})\subseteq\{0,1,\dots,n-1\}$ だから $\max I(\mathtt{yMax})=n-1$。

**(4)**
まず $y=0$ では $Y(0)=0$ より $0\in I(0)$ であり、$I(0)\subseteq\mathbb{Z}_{\ge0}$ なので $\min I(0)=0$。
よって $\ell(0):=0=\min I(0)$。

以下 $1\le y\le \mathtt{yMax}$ とする。$u:=my-d$ とおくと、$y\ge1$ と $0\le d<m$ より $u\ge m-d\ge1$ である。

$$
\ell:=\left\lceil\frac{u}{c}\right\rceil
$$

とおく（これが定義した $\ell(y)$ である）。

**(4-a) 天井 $\Rightarrow$ 床 への恒等式**

$c>0$ と $u\in\mathbb{Z}$ に対し

$$
\left\lceil\frac{u}{c}\right\rceil=\left\lfloor\frac{u+c-1}{c}\right\rfloor
$$

が成り立つ。実際、

$$
q:=\left\lfloor\frac{u+c-1}{c}\right\rfloor
$$

とおけば $cq\le u+c-1<c(q+1)$ より $c(q-1)<u\le cq$、
したがって $q=\left\lceil\frac{u}{c}\right\rceil$。

よって

$$
\ell(y)=\left\lceil\frac{my-d}{c}\right\rceil
:=\left\lfloor\frac{my-d+c-1}{c}\right\rfloor
$$

が従う。

**(4-b) 隣接関係 $\ell(y)=r(y-1)+1$**

(2) の定義より

$$
r(y-1)=\left\lceil\frac{m((y-1)+1)-d}{c}\right\rceil-1
:=\left\lceil\frac{my-d}{c}\right\rceil-1,
$$

したがって $\ell(y)=r(y-1)+1$ が成り立つ。

**(4-c) $\ell(y)\in I(y)$（つまり $Y(\ell)=y$）**

$\ell=\left\lceil\frac{u}{c}\right\rceil$ の定義から

$$
c(\ell-1)<u\le c\ell
$$

が成り立つ。両辺に $d$ を足すと

$$
c(\ell-1)+d<my,\qquad c\ell+d\ge my.
$$

ゆえに

$$
Y(\ell-1)=\left\lfloor\frac{c(\ell-1)+d}{m}\right\rfloor\le y-1,
\qquad
Y(\ell)=\left\lfloor\frac{c\ell+d}{m}\right\rfloor\ge y.
$$

一方 (1) より $Y(\ell)\le Y(\ell-1)+1$ なので

$$
y\le Y(\ell)\le Y(\ell-1)+1\le (y-1)+1=y
$$

が従い、結局 $Y(\ell)=y$ である。したがって $\ell\in I(y)$。

さらに $y\le \mathtt{yMax}=Y(n-1)$ より

$$
my\le c(n-1)+d \ \Rightarrow\ \frac{my-d}{c}\le n-1 \ \Rightarrow\ \ell=\left\lceil\frac{my-d}{c}\right\rceil\le n-1
$$

なので $0\le \ell<n$ が成り立ち、$I(y)$ の定義域にも入っている。

**(4-d) 最小性（$\ell(y)=\min I(y)$）**

任意に $x<\ell$ をとると $x\le \ell-1$ であり、単調性より

$$
Y(x)\le Y(\ell-1)\le y-1
$$

なので $Y(x)\ne y$、すなわち $x\notin I(y)$。
よって $\ell=\min I(y)$ が従う。□

#### a が 0 以上の場合の小問題への縮約

#### 補題（a が 0 以上の場合の縮約公式）

正規化後に $\mathtt{yMax}\ge1$ かつ $a\ge0$ とする。このとき

$$
\max_{0\le x<n}\left(ax+b\left\lfloor\frac{cx+d}{m}\right\rfloor\right)
=\max\Bigl(a(n-1)+b\cdot\mathtt{yMax},\ \mathtt{mwf}(0,\mathtt{yMax},c,b,a,m,m-d-1)\Bigr)
$$

が成り立つ。

**証明**

$a\ge0$ のとき、各段 $I(y)$ 上では $Y(x)=y$ が一定であるから

$$
\phi(x)=ax+bY(x)=ax+by\qquad (x\in I(y))
$$

となり、$x\mapsto ax+by$ は単調非減少である。したがって

$$
\max_{x\in I(y)} \phi(x)=a\,r(y)+by
$$

を得る。[補題（階段区間の端点公式）](#補題（階段区間の端点公式）)(2) より、$0\le y<\mathtt{yMax}$ に対して

$$
r(y)=\left\lfloor\frac{m(y+1)-d-1}{c}\right\rfloor
=\left\lfloor\frac{my+(m-d-1)}{c}\right\rfloor
$$

なので

$$
\begin{aligned}
\max_{0\le x<n}\bigl(ax+bY(x)\bigr)
&=\max\Bigl( a(n-1)+b\cdot\mathtt{yMax},\ \max_{\substack{y\in\mathbb{Z}_{\ge0}\\0\le y<\mathtt{yMax}}}\ \max_{x\in I(y)}\bigl(ax+by\bigr)\Bigr) \\
&=\max\Bigl( a(n-1)+b\cdot\mathtt{yMax},\ \max_{\substack{y\in\mathbb{Z}_{\ge0}\\0\le y<\mathtt{yMax}}}\left(a\left\lfloor\frac{my+(m-d-1)}{c}\right\rfloor+by\right)\Bigr).
\end{aligned}
$$

$0\le y<\mathtt{yMax}$ では $r(y)\in[0,n)$ であり、$y=\mathtt{yMax}$ は $x=n-1$ に対応して第1項で既に取り込まれている。よって $x\leftarrow y$ と読み替えると

$$
\max_{0\le x<\mathtt{yMax}}\left(bx+a\left\lfloor\frac{mx+(m-d-1)}{c}\right\rfloor\right)
=\mathtt{mwf}(0,\mathtt{yMax},c,b,a,m,m-d-1)
$$

となり、主張を得る。□

#### a が負の場合の小問題への縮約

#### 補題（a が負の場合の縮約公式）

正規化後に $\mathtt{yMax}\ge1,\ a<0,\ b>0$ とする。このとき

$$
\max_{0\le x<n}\left(ax+b\left\lfloor\frac{cx+d}{m}\right\rfloor\right)
=\max\Bigl(0,\ a+b+\mathtt{mwf}(0,\mathtt{yMax},c,b,a,m,m-d-1)\Bigr)
$$

が成り立つ。

**証明**

$a<0$ のとき、各段 $I(y)$ 上では $Y(x)=y$ が一定であるから

$$
\phi(x)=ax+bY(x)=ax+by\qquad (x\in I(y))
$$

となり、$x\mapsto ax+by$ は単調非増加である。したがって

$$
\max_{x\in I(y)} \phi(x)=a\,\ell(y)+by
$$

である。[補題（階段区間の端点公式）](#補題（階段区間の端点公式）)より

$$
\ell(y)=r(y-1)+1\qquad (1\le y\le \mathtt{yMax})
$$

なので

$$
\begin{aligned}
a\,\ell(y)+by
&=a\,(r(y-1)+1)+by \\
&=(a+b)+\bigl(a\,r(y-1)+b(y-1)\bigr)
\end{aligned}
$$

から

$$
\max_{1\le y\le \mathtt{yMax}}\bigl(a\,\ell(y)+by\bigr)
=(a+b)+\max_{0\le y<\mathtt{yMax}}\bigl(a\,r(y)+by\bigr)
$$

を得る。

ここで参照する $1\le y\le \mathtt{yMax}$ の $\ell(y)$ は常に $[0,n)$ 内にある。実際、[補題（階段区間の端点公式）](#補題（階段区間の端点公式）)(4) より $\ell(y)=\min I(y)$ であり、$I(y)$ の定義から $0\le \ell(y)<n$ が従う。

よって（端点 $x=0$ は $\phi(0)=0$ として別に候補に含めることで）

$$
\max_{0\le x<n} \phi(x)=\max\Bigl(0,\ (a+b)+\max_{0\le y<\mathtt{yMax}}\bigl(a\,r(y)+by\bigr)\Bigr)
$$

となる。さらに $\max_{0\le y<\mathtt{yMax}}(a\,r(y)+by)$ は前段と同様に$x\leftarrow y$ と読み替えることで

$$
\mathtt{mwf}(0,\mathtt{yMax},c,b,a,m,m-d-1)
$$

に一致する。□

以上より、$a<0$ の場合は定数補正

$$
\mathtt{sumAcc}\leftarrow \mathtt{sumAcc}+a+b
$$

を加えた上で、

$$
(n,m,a,b,c,d)\leftarrow (\mathtt{yMax},\ c,\ b,\ a,\ m,\ m-d-1)
$$

という $a\ge0$ の場合と同じ更新則で同型の小問題へ縮約できる。

#### 分岐再帰の結論

#### 補題（`mwf` の分岐再帰）

$0<n,\ 0<m,\ 0\le c,d<m$ とする。まず

$$
\mathtt{yMax}:=\max_{x\in\mathbb{Z}_{\ge0},\ 0\le x<n}\left\lfloor\frac{cx+d}{m}\right\rfloor
=\left\lfloor\frac{c(n-1)+d}{m}\right\rfloor
$$

とおく。さらに $\mathtt{yMax}\ge1$ の場合に限り

$$
\mathtt{mwfY}:=\mathtt{mwf}(0,\ \mathtt{yMax},\ c,\ b,\ a,\ m,\ m-d-1)
$$

とおく（$\mathtt{yMax}=0$ の場合は $\mathtt{mwfY}$ を用いない）。このとき

$$
\begin{aligned}
&\mathtt{mwf}(0,n,m,a,b,c,d) \\
&=\max_{x\in\mathbb{Z}_{\ge0},\ 0\le x<n}\left(a x + b\left\lfloor\frac{c x + d}{m}\right\rfloor\right) \\
&=\begin{cases}
\max(0,\ a(n-1)+b\cdot \mathtt{yMax}),
& \mathtt{yMax}=0\ \vee\ ab\ge 0, \\
\max\bigl(a(n-1)+b\cdot \mathtt{yMax},\ \mathtt{mwfY}\bigr),
& \mathtt{yMax}\ge 1,\ a>0,\ b<0, \\
\max\bigl(0,\ a+b+\mathtt{mwfY}\bigr),
& \mathtt{yMax}\ge 1,\ a<0,\ b>0.
\end{cases}
\end{aligned}
$$

が成り立つ。

**証明**

$\mathtt{yMax}$ の公式は[補題（端点での階段関数最大値）](#補題（端点での階段関数最大値）)による。

まず $\mathtt{yMax}=0$ または $ab\ge0$ の場合、[補題（端点で終了できる条件）](#補題（端点で終了できる条件）)より最大値は端点で確定する。正規化下では $Y(0)=0$ なので $x=0$ での値は $0$、また $x=n-1$ での値は$a(n-1)+b\cdot\mathtt{yMax}$ だから、第1分岐を得る。

つぎに $\mathtt{yMax}\ge1,\ a>0,\ b<0$ なら[補題（$a$ が 0 以上の場合の縮約公式）](#補題（a-が-0-以上の場合の縮約公式）)を適用でき、$\mathtt{mwfY}$ の定義から第2分岐を得る。

最後に $\mathtt{yMax}\ge1,\ a<0,\ b>0$ なら[補題（$a$ が負の場合の縮約公式）](#補題（a-が負の場合の縮約公式）)を適用でき、$\mathtt{mwfY}$ の定義から第3分岐を得る。□

#### `mwf` の計算量

本節の更新は、非停止が続く区間で

$$
(c,m)\mapsto (m,\;c\bmod m)
$$

という Euclid 型の反復になる。とくに

$$
0<c<m
$$

なら最初の 1 回は単純交換 $(c,m)\mapsto(m,c)$、一方で

$$
0<m\le c
$$

なら 1 回目から上式の Euclid 反復である。

ここで Fibonacci 関数 $\mathrm{fib}:\mathbb{N}\to\mathbb{N}$ を

$$
\mathrm{fib}(0)=0,\qquad \mathrm{fib}(1)=1,\qquad
\mathrm{fib}(n+2)=\mathrm{fib}(n+1)+\mathrm{fib}(n)\ (n\ge0)
$$

で定義する。また

$$
\varphi:=\frac{1+\sqrt5}{2},\qquad
\psi:=\frac{1-\sqrt5}{2}=-\frac{1}{\varphi},\qquad
1<\varphi<2,\qquad
-1<\psi<0
$$

とし、Binet 公式

$$
\mathrm{fib}(t)=\frac{\varphi^t-\psi^t}{\sqrt5}
$$

を用いる。

#### 補題（対数床と Fibonacci 区間の同値）

任意の $n,k\in\mathbb{N}$ について

$$
k=\left\lfloor \log_\varphi\!\bigl(\sqrt5\,(n+\tfrac12)\bigr)\right\rfloor
\iff
\mathrm{fib}(k)\le n<\mathrm{fib}(k+1)
$$

が成り立つ。

**証明**

$t\in\mathbb{N}$ に対する前提の Binet 公式を

$$
\sqrt5\,\mathrm{fib}(t)=\varphi^t-\psi^t
\quad\Longleftrightarrow\quad
\varphi^t=\sqrt5\,\mathrm{fib}(t)+\psi^t
$$

と変形する。さらに $-1<\psi<0$ なので $-1\le\psi^t\le 1$、したがって

$$
-\frac{\sqrt5}{2}<-1\le\psi^t\le1<\frac{\sqrt5}{2}
$$

である。これを上式に加えると

$$
\sqrt5\!\left(\mathrm{fib}(t)-\tfrac12\right)<\varphi^t<
\sqrt5\!\left(\mathrm{fib}(t)+\tfrac12\right)\qquad(t\in\mathbb{N})
$$

を得る。

($\Rightarrow$)\;$k=\lfloor\log_\varphi(\sqrt5(n+\tfrac12))\rfloor$ なら

$$
\varphi^k\le \sqrt5\!\left(n+\tfrac12\right)<\varphi^{k+1}.
$$

左側と上の評価（$t=k$）から

$$
\sqrt5\!\left(n+\tfrac12\right)>
\varphi^k>\sqrt5\!\left(\mathrm{fib}(k)-\tfrac12\right)
$$

となるので $\mathrm{fib}(k)\le n$。右側と上の評価（$t=k+1$）から

$$
\sqrt5\!\left(n+\tfrac12\right)<\varphi^{k+1}<
\sqrt5\!\left(\mathrm{fib}(k+1)+\tfrac12\right)
$$

となるので $n<\mathrm{fib}(k+1)$。よって $\mathrm{fib}(k)\le n<\mathrm{fib}(k+1)$。

($\Leftarrow$)\;$\mathrm{fib}(k)\le n<\mathrm{fib}(k+1)$ を仮定する。$n$ は整数なので

$$
\mathrm{fib}(k)+\tfrac12\le n+\tfrac12\le \mathrm{fib}(k+1)-\tfrac12.
$$

したがって上の評価（$t=k,k+1$）より

$$
\varphi^k<\sqrt5\!\left(\mathrm{fib}(k)+\tfrac12\right)\le
\sqrt5\!\left(n+\tfrac12\right)
$$

および

$$
\sqrt5\!\left(n+\tfrac12\right)\le
\sqrt5\!\left(\mathrm{fib}(k+1)-\tfrac12\right)<\varphi^{k+1}
$$

を得る。ゆえに

$$
\varphi^k\le \sqrt5\!\left(n+\tfrac12\right)<\varphi^{k+1}.
$$

$\log_\varphi$ の単調増加性から

$$
k\le \log_\varphi\!\bigl(\sqrt5(n+\tfrac12)\bigr)<k+1
$$

であり、

$$
k=\left\lfloor \log_\varphi\!\bigl(\sqrt5(n+\tfrac12)\bigr)\right\rfloor
$$

が従う。□

#### 補題（Fibonacci の対数上界）

任意の正整数 $m_0$ について

$$
m_0<\mathrm{fib}\!\left(\left\lfloor\log_{\varphi}m_0\right\rfloor+3\right)
$$

が成り立つ。

**証明**

$m:=\lfloor\log_{\varphi}m_0\rfloor$ とおく。床関数の性質から

$$
\log_{\varphi}m_0<m+1
$$

なので

$$
m_0<\varphi^{m+1}
$$

を得る。

また前提の Binet 公式から

$$
\begin{aligned}
\varphi\,\mathrm{fib}(t+1)+\mathrm{fib}(t)
&=\frac{\varphi^{t+2}-\varphi\psi^{t+1}+\varphi^t-\psi^t}{\sqrt5} \\
&=\frac{\varphi^{t+2}+\varphi^t-(\varphi\psi+1)\psi^t}{\sqrt5} \\
&=\frac{\varphi^{t+2}+\varphi^t}{\sqrt5} \\
&=\frac{\varphi^t(\varphi^2+1)}{\sqrt5} \\
&=\frac{\varphi^t(\sqrt5\,\varphi)}{\sqrt5}
&& (\varphi^2+1=\sqrt5\,\varphi) \\
&=\varphi^{t+1}
\end{aligned}
$$

を得る。したがって

$$
\varphi^{t+1}=\varphi\,\mathrm{fib}(t+1)+\mathrm{fib}(t)
$$

であり、$1<\varphi<2$ から

$$
\begin{aligned}
\varphi^{m+1}
&=\varphi\,\mathrm{fib}(m+1)+\mathrm{fib}(m) \\
&<2\,\mathrm{fib}(m+1)+\mathrm{fib}(m) \\
&=\mathrm{fib}(m+3)
\end{aligned}
$$

である。以上より

$$
m_0<\varphi^{m+1}<\mathrm{fib}(m+3)
=\mathrm{fib}\!\left(\left\lfloor\log_{\varphi}m_0\right\rfloor+3\right)
$$

が成り立つ。□

#### 補題（Euclid 反復の古典的評価）

任意の初期値 $c_0\in\mathbb{Z}_{\ge0},\ m_0\in\mathbb{Z}_{\ge0}$ に対し、Euclid 反復を

$$
(c_{i+1},m_{i+1})=
\begin{cases}
(m_i,\;c_i\bmod m_i) & (m_i>0),\\
(c_i,\;0) & (m_i=0)
\end{cases}
$$

で定め、停止時刻を

$$
\tau:=\min\{\,t\in\mathbb{N}\mid m_t=0\,\}
$$

とする。このとき任意の $k\in\mathbb{N}$ について

$$
m_0<\mathrm{fib}(k+2)\ \Longrightarrow\ \tau\le k
$$

が成り立つ。

**証明**

対偶を示す。すなわち

$$
\tau>k \Rightarrow m_0\ge \mathrm{fib}(k+2)
$$

を示せば十分である。以下、$\tau>k$ と仮定すると

$$
m_0,m_1,\dots,m_k>0
$$

である。ここで $k=0$ のときは $\tau>0$ より $m_0>0$ であり、$m_0\ge1=\mathrm{fib}(2)=\mathrm{fib}(k+2)$ となるので対偶主張は成り立つ。以下 $k\ge1$ とする。

**(A) Euclid 不等式 $m_{i-1}\ge m_i+m_{i+1}$**

まず $1\le i\le k-1$ を任意にとる。定義から $c_i=m_{i-1}$ なので

$$
m_{i+1}=c_i\bmod m_i=(m_{i-1})\bmod m_i,\qquad 0\le m_{i+1}<m_i
$$

である。また $i\ge1$ では $m_i=c_{i-1}\bmod m_{i-1}$ であり、$m_{i-1}>0$ だから $m_i<m_{i-1}$ が成り立つ。よって

$$
m_{i-1}=q_i m_i+m_{i+1},\qquad q_i:=\left\lfloor\frac{m_{i-1}}{m_i}\right\rfloor\in\mathbb{N}
$$

と書け、しかも $m_{i-1}>m_i$ から $q_i\ge1$ である。従って

$$
m_{i-1}=q_i m_i+m_{i+1}\ \ge\ m_i+m_{i+1}\qquad(1\le i\le k-1)
$$

を得る。

**(B) 後ろ向き帰納で Fibonacci 下界**

以下

$$
m_{k-s}\ge \mathrm{fib}(s+2)\qquad(0\le s\le k)
$$

を示す。$s=0$ は $m_k>0$ より $m_k\ge1=\mathrm{fib}(2)$。$s=1$ は、定義から

$$
m_k=c_{k-1}\bmod m_{k-1},\qquad 0\le m_k<m_{k-1}
$$

であり、かつ $m_k>0$ なので $m_{k-1}\ge2=\mathrm{fib}(3)$。

$1\le s\le k-1$ とし、$m_{k-s}\ge\mathrm{fib}(s+2)$ および $m_{k-s+1}\ge\mathrm{fib}(s+1)$ を仮定する。上の不等式を $i=k-s$ に適用すると

$$
\begin{aligned}
m_{k-s-1}
&\ge m_{k-s}+m_{k-s+1} \\
&\ge \mathrm{fib}(s+2)+\mathrm{fib}(s+1) \\
&=\mathrm{fib}(s+3)
\end{aligned}
$$

より帰納が閉じる。したがって $s=k$ で

$$
m_0\ge \mathrm{fib}(k+2)
$$

を得る。以上より

$$
\tau>k \Rightarrow m_0\ge \mathrm{fib}(k+2)
$$

が示せたので

$$
m_0<\mathrm{fib}(k+2)\Rightarrow \tau\le k
$$

が従う。□

#### 定義（Fib 添字 g(m0)）

$m_0\in\mathbb{N}$ に対し

$$
g(m_0):=\mathtt{greatestFib}(m_0),\qquad
\mathrm{fib}(g(m_0))\le m_0<\mathrm{fib}(g(m_0)+1)
$$

で定める（すなわち $\mathrm{fib}(k)\le m_0$ を満たす最大の $k$）。

#### 補題（`greatestFib` による停止時刻上界）

$m_0\in\mathbb{N},\ m_0>0$ とし、$g(m_0)$ は上の定義で定める。このとき

$$
g(m_0)=\left\lfloor \log_\varphi\!\bigl(\sqrt5\,(m_0+\tfrac12)\bigr)\right\rfloor
$$

$$
g(m_0)\le \left\lfloor\log_{\varphi}m_0\right\rfloor+2
$$

$$
\tau\le g(m_0)-1\le \left\lfloor\log_{\varphi}m_0\right\rfloor+1
$$

が成り立つ。特に Euclid 反復は高々 $g(m_0)-1$ 回で停止する。

**証明**

[補題（対数床と Fibonacci 区間の同値）](#補題（対数床と-fibonacci-区間の同値）)を $n=m_0,\ k=g(m_0)$ に適用すると

$$
g(m_0)=\left\lfloor \log_\varphi\!\bigl(\sqrt5\,(m_0+\tfrac12)\bigr)\right\rfloor
$$

を得る。さらに[補題（Fibonacci の対数上界）](#補題（fibonacci-の対数上界）)より

$$
g(m_0)\le \left\lfloor\log_{\varphi}m_0\right\rfloor+2
$$

である。

また[補題（Euclid 反復の古典的評価）](#補題（euclid-反復の古典的評価）)より、任意の $k\in\mathbb{N}$ で

$$
m_0<\mathrm{fib}(k+2)\Rightarrow \tau\le k
$$

が成り立つ。ここで $m_0>0$ より $m_0\ge1=\mathrm{fib}(1)$ なので $g(m_0)\ge1$、従って $k:=g(m_0)-1\in\mathbb{N}$ と置ける。定義から

$$
m_0<\mathrm{fib}(g(m_0)+1)=\mathrm{fib}\bigl((g(m_0)-1)+2\bigr)
$$

であるから、上の含意に $k=g(m_0)-1$ を代入して

$$
\tau\le g(m_0)-1
$$

を得る。最後に $\tau\le g(m_0)-1$ と $g(m_0)\le \left\lfloor\log_{\varphi}m_0\right\rfloor+2$ を合わせれば

$$
\tau\le \left\lfloor\log_{\varphi}m_0\right\rfloor+1
$$

が従う。□

一方、実装では簡潔な安全上界として `step` の反復回数を

$$
\mathtt{stepBoundOfM}(m_0)
:=g(m_0)
:=\left\lfloor \log_\varphi\!\bigl(\sqrt5\,(m_0+\tfrac12)\bigr)\right\rfloor
\le \left\lfloor\log_{\varphi}m_0\right\rfloor+2
$$

で抑えられ、$\mathcal{O}(\log m_0)$ である。`step` 1 回を $\mathcal{O}(1)$ とみなせば、`mwf_iter` と `mwfLr_iter` はともに $\mathcal{O}(\log m_0)$（各局面の分母 $m$ に対しては $\mathcal{O}(\log m)$）となる。

#### 閾値判定（`mwf_leq`）

二分探索では最大値そのものではなく

$$
\mathtt{mwf}(l,r,m,a,b,c,d)\le \theta
$$

の真偽のみが必要である。この場合、$\phi(x)$ の代わりに $\phi(x)-\theta$ を扱い、$\mathtt{sumAcc}$ や端点評価が正になった時点で直ちに `False` を返すことで早期終了できる（この方針をとる `mwf_leq` の実装は[divapprox2](#divapprox2)にて提示する）。

### `floor_prod` に基づく `mwf`

[AtCoder Library (ACL)](https://github.com/atcoder/ac-library) の `floor_sum` をモノイド積に拡張した手法（以下 `floor_prod` と呼ぶ）を用いると、`mwf` も同様の枠組みで実装できる。この枠組みの背景は [参考文献 4,5,6](#参考文献) に基づく。なお、可換モノイドが可換則 $\forall a,b\in\mathcal{M},\ a\cdot b=b\cdot a$ を仮定するのに対し、非可換モノイドでは可換則を仮定しない。本稿では積の順序を固定して扱うため、可換・非可換のいずれにも同様に適用できる。
実際、後に扱う `MwfElem` のモノイドは一般に非可換である。

**格子路による見方**

直線 $y=(ax+b)/m$ の下側で $(0,0)$ から $(n,\lfloor(an+b)/m\rfloor)$ へ$y$ 方向優先で進む格子路を考える。$x$ 方向へ 1 歩進むたびに元 $x$ を、$y$ 方向へ 1 歩進むたびに元 $y$ を経路順に右から掛ける（累積値 $\mathrm{acc}$ を $\mathrm{acc}\leftarrow \mathrm{acc}\cdot(\cdots)$ で更新する）と、経路に対応する積が定義される。`floor_prod` はこの積を `floor_sum` 型の再帰で計算する手続きである。

#### 定義（`floor_prod`）

モノイド $\mathcal{M}$、積 $\cdot$、単位元 $e$ について、以下を仮定する。
- 積 $\cdot$ は $\mathcal{M}$ 上の二項演算であり、任意の $u,v,w\in\mathcal{M}$ に対して結合則 $(u\cdot v)\cdot w=u\cdot(v\cdot w)$ が成り立つ。
- $e\in\mathcal{M}$ は単位元であり、任意の $u\in\mathcal{M}$ に対して $e\cdot u=u\cdot e=u$ が成り立つ。
- 任意の $u\in\mathcal{M},\ k\in\mathbb{Z}_{\ge0}$ に対し、$u^k$ はモノイド積による $k$ 乗を表す。とくに $u^0=e$ とする。

元 $x,y\in\mathcal{M}$ と整数 $n\ge0,m>0,a\ge0,b\ge0$ をとる。したがって $x^0=e,\ y^0=e$ が成り立つ。

$$
\mathtt{floor\_prod}(n,m,a,b;e,x,y)
:=y^{\lfloor b/m\rfloor}\cdot
\prod_{i=0}^{n-1}\left(x\cdot y^{\left\lfloor\frac{a(i+1)+b}{m}\right\rfloor-\left\lfloor\frac{ai+b}{m}\right\rfloor}\right)
$$

（積は左から右へ計算）で定義する。これは `floor_sum` の格子路表現に対応する。

#### 補題（`floor_prod` の正規化）

整数 $n\ge0,m>0,a\ge0,b\ge0$ と $x,y\in\mathcal{M}$ をとる。

$$
q_a:=\left\lfloor\frac{a}{m}\right\rfloor,\quad r_a:=a\bmod m,\qquad
q_b:=\left\lfloor\frac{b}{m}\right\rfloor,\quad r_b:=b\bmod m
$$

とおくと

$$
\mathtt{floor\_prod}(n,m,a,b;e,x,y)
=
y^{q_b}\cdot
\mathtt{floor\_prod}(n,m,r_a,r_b;e,\ x\cdot y^{q_a},\ y)
$$

が成り立つ。とくに右辺では $0\le r_a,r_b<m$ である。

**証明**

$a=q_am+r_a,\ b=q_bm+r_b$ と書けるので、任意の $i\in\{0,\dots,n\}$ で

$$
\left\lfloor\frac{ai+b}{m}\right\rfloor
=
q_ai+q_b+\left\lfloor\frac{r_ai+r_b}{m}\right\rfloor
$$

が成り立つ。したがって指数差は

$$
\left\lfloor\frac{a(i+1)+b}{m}\right\rfloor-\left\lfloor\frac{ai+b}{m}\right\rfloor
=
q_a+
\left(
\left\lfloor\frac{r_a(i+1)+r_b}{m}\right\rfloor-\left\lfloor\frac{r_ai+r_b}{m}\right\rfloor
\right)
$$

となる。定義に代入すると

$$
y^{\lfloor b/m\rfloor}=y^{q_b},\qquad
x\cdot y^{q_a+\left(\left\lfloor\frac{r_a(i+1)+r_b}{m}\right\rfloor-\left\lfloor\frac{r_ai+r_b}{m}\right\rfloor\right)}
=(x\cdot y^{q_a})\cdot
y^{\left(\left\lfloor\frac{r_a(i+1)+r_b}{m}\right\rfloor-\left\lfloor\frac{r_ai+r_b}{m}\right\rfloor\right)}
$$

より主張の等式を得る。□

#### 補題（`floor_prod` の再帰式）

補題（`floor_prod` の正規化）により一般の $a,b\ge0$ から $0\le a,b<m$ に正規化できる。以下、正規化後を考え、

$$
y_{\max}:=\left\lfloor\frac{an+b}{m}\right\rfloor,\qquad
r:=(an+b)\bmod m
$$

とおく。`floor_prod` の引数に現れる単位元 $e$ は固定なので、以下では省略して $\mathtt{floor\_prod}(n,m,a,b;x,y)$ と書く。このとき

$$
\begin{aligned}
&\mathtt{floor\_prod}(n,m,a,b;x,y)\\
&=
\begin{cases}
x^n, & y_{\max}=0,\\
\mathtt{floor\_prod}\!\left(y_{\max}-1,\ a,\ m,\ m+a-b-1;\ y,\ x\right)\cdot y\cdot x^{\left\lfloor\frac{r}{a}\right\rfloor},
& y_{\max}\ge 1.
\end{cases}
\end{aligned}
$$

が成り立つ（第2分岐は $a>0$ の場合に現れ、$a=0$ では常に第1分岐で停止する）。

**証明**

定義の各因子で現れる $y$ の指数差

$$
\delta_i:=\left\lfloor\frac{a(i+1)+b}{m}\right\rfloor-\left\lfloor\frac{ai+b}{m}\right\rfloor
$$

を考える。$y_{\max}=0$ なら $an+b<m$ なので全て $\delta_i=0$、ゆえに積は $x^n$ に一致する。$y_{\max}\ge 1$ の場合は、[AtCoder Library (ACL)](https://github.com/atcoder/ac-library) の `floor_sum` と同型の格子路再帰で $(n,m,a,b)$ が

$$
\left(y_{\max}-1,\ a,\ m,\ m+a-b-1\right)
$$

に移り、$x,y$ が交換され、境界項として $y\cdot x^{\lfloor r/a\rfloor}$ が付く。これをそのままモノイド積に移せば式を得る。□

**再帰（反復）更新**

以下で最初に行う $p,q$ による更新は、補題（`floor_prod` の正規化）を1回適用して $a,b$ を $0\le a,b<m$ に正規化する処理に対応する。

実装では、再帰の外側に出る積を $\mathtt{pre},\mathtt{suf}$ に保持して while ループへ書き換える。コード上の初期化は

$$
c\leftarrow\left\lfloor\frac{an+b}{m}\right\rfloor,\qquad
\mathtt{pre}\leftarrow e,\quad \mathtt{suf}\leftarrow e
$$

である。ループ本体では、まず

$$
p=\left\lfloor\frac{a}{m}\right\rfloor,\ a\leftarrow a\bmod m,\quad
x\leftarrow x\cdot y^p,
$$

$$
q=\left\lfloor\frac{b}{m}\right\rfloor,\ b\leftarrow b\bmod m,\quad
\mathtt{pre}\leftarrow \mathtt{pre}\cdot y^q,
$$

$$
c\leftarrow c-(pn+q)
$$

と更新する。このとき不変量

$$
c=\left\lfloor\frac{an+b}{m}\right\rfloor
$$

が保たれる（実装中の `assert` に対応）。$c=0$ で停止し

$$
\mathtt{pre}\cdot x^n\cdot \mathtt{suf}
$$

を返す。$c>0$（したがって $a>0$）なら

$$
d:=\left\lfloor\frac{mc-b-1}{a}\right\rfloor+1
=n-\left\lfloor\frac{(an+b)\bmod m}{a}\right\rfloor,
$$

$$
\mathtt{suf}\leftarrow y\cdot x^{\,n-d}\cdot \mathtt{suf},
$$

$$
(n,m,a,b,c,x,y)\leftarrow(c-1,\ a,\ m,\ m-b-1+a,\ d,\ y,\ x)
$$

と更新する。これが `floor_prod` 実装の while ループそのものである。

**`mwf` への写像**

$$
\phi(i):=ai+b\left\lfloor\frac{ci+d}{m}\right\rfloor\qquad(0\le i<n)
$$

の最大値と $\arg\max$ を同時に得るには、$f(i):=\left\lfloor\dfrac{ci+d}{m}\right\rfloor$ とおき、上の `floor_prod` を $(n,m,c,d)$ に対して適用する。このとき、モノイド要素を

$$
\begin{aligned}
X&=(\mathtt{sum}=a,\ \mathtt{best}=0,\ \mathtt{dx}=1,\ \mathtt{arg}=0),\\
Y&=(\mathtt{sum}=b,\ \mathtt{best}=0,\ \mathtt{dx}=0,\ \mathtt{arg}=0)
\end{aligned}
$$

と定める（実装では `MwfElem`）。コード上の呼び出しは

$$
\mathtt{res}
:=\mathtt{floor\_prod}\bigl(n,m,c,d,\ \mathtt{MwfElem}(0),\ X,\ Y\bigr)
$$

であり、`argmax_weighted_floor` では冒頭で `assert n > 0 and m > 0`、末尾で `assert res.dx > 0` を置いている。

#### 定義（`MwfElem` の意味づけ）

まず型

$$
\operatorname{Option}(U):=\{\mathrm{None}\}\cup\{\mathrm{Some}(u)\mid u\in U\}
$$

$$
\mathcal{T}_{\mathrm{mwf}}:=
\mathbb{Z}\times\operatorname{Option}(\mathbb{Z})\times\mathbb{N}\times\operatorname{Option}(\mathbb{N})
$$

を定める。以下では語 $W$ の要約データを $(S,B,D,A)\in\mathcal{T}_{\mathrm{mwf}}$ と書く。

すなわち

$$
S\in\mathbb{Z},\qquad
B\in\operatorname{Option}(\mathbb{Z}),\qquad
D\in\mathbb{N},\qquad
A\in\operatorname{Option}(\mathbb{N})
$$

次に $X,Y$ からなる語 $W$ に対し、重みを

$$
w:\{X,Y\}\to\mathbb{Z},\qquad w(X)=a,\ w(Y)=b
$$

とする。$W$ に含まれる $X$ の個数を $D(W)$ とし、$t$ 番目の $X$（$0\le t<D(W)$）の直前までの重み和を $P_t(W)$ とおく。このとき

$$
P_t(W)\in\mathbb{Z}\qquad(0\le t<D(W))
$$

である。さらに

$$
S(W):=\sum_{u\in W} w(u)
$$

$$
B(W):=
\begin{cases}
\mathrm{Some}\!\left(B^{\ast}(W)\right), & D(W)>0,\\
\mathrm{None}, & D(W)=0,
\end{cases}
$$

$$
A(W):=
\begin{cases}
\mathrm{Some}\!\left(A^{\ast}(W)\right), & D(W)>0,\\
\mathrm{None}, & D(W)=0,
\end{cases}
$$

$$
B^{\ast}(W):=\max_{0\le t<D(W)} P_t(W)\qquad(D(W)>0)
$$

$$
A^{\ast}(W):=\min\{\,t\mid P_t(W)=B^{\ast}(W)\,\}\qquad(D(W)>0)
$$

で定める。

**実装との対応**

実装では常に整数 4 つ

$$
\mathtt{MwfElem}=(\mathtt{sum},\mathtt{best},\mathtt{dx},\mathtt{arg})
$$

を保持し、$\mathtt{dx}=D(W)$ とする。$D(W)>0$ のときのみ $(\mathtt{best},\mathtt{arg})=(B^{\ast}(W),A^{\ast}(W))$ を保持し、

$$
\bigl(B(W),A(W)\bigr)=\bigl(\mathrm{Some}(\mathtt{best}),\mathrm{Some}(\mathtt{arg})\bigr)
$$

と解釈する。$D(W)=0$ のときは数学的には

$$
\bigl(B(W),A(W)\bigr)=\bigl(\mathrm{None},\mathrm{None}\bigr)
$$

であり、実装例では便宜上

$$
(\mathtt{best},\mathtt{arg})=(0,0)
$$

を代入して保持するが演算に用いない。

#### 補題（`MwfElem` の積更新）

定義（`MwfElem` の意味づけ）で導入した型 $\mathcal{T}_{\mathrm{mwf}}$ を用いる。

語の連結 $W=W_1W_2$ に対し

$$
Z_i:=\bigl(S(W_i),B(W_i),D(W_i),A(W_i)\bigr)
\in\mathcal{T}_{\mathrm{mwf}}
\qquad(i=1,2)
$$

とおく。また

$$
Z:=\bigl(S(W),B(W),D(W),A(W)\bigr)
\in\mathcal{T}_{\mathrm{mwf}}
$$

とおく。語連結に対応する `MwfElem` の積を $\cdot$ と書けば

$$
Z=Z_1\cdot Z_2
$$

である。したがって

$$
S(W)=S(W_1)+S(W_2),\qquad D(W)=D(W_1)+D(W_2).
$$

特に $D(W_1)=D(W_2)=0$ のときは $D(W)=0$ であり、定義により $\bigl(B(W),A(W)\bigr)=\bigl(\mathrm{None},\mathrm{None}\bigr)$ である（実装値は演算に用いない）。
一方で $D(W)>0$ の場合、定義の $\mathrm{Some}$ の中身 $B^{\ast}(W),A^{\ast}(W)$ は次で与えられる。

$$
L:=B^{\ast}(W_1),\qquad R:=S(W_1)+B^{\ast}(W_2)\qquad(D(W_1),D(W_2)>0)
$$

$$
\bigl(B^{\ast}(W),A^{\ast}(W)\bigr)=
\begin{cases}
\bigl(B^{\ast}(W_1),A^{\ast}(W_1)\bigr), & D(W_2)=0<D(W_1),\\
\bigl(S(W_1)+B^{\ast}(W_2),A^{\ast}(W_2)\bigr), & D(W_1)=0<D(W_2),\\
\bigl(L,A^{\ast}(W_1)\bigr), & D(W_1),D(W_2)>0,\ L\ge R,\\
\bigl(R,\,D(W_1)+A^{\ast}(W_2)\bigr), & D(W_1),D(W_2)>0,\ L<R.
\end{cases}
$$

が成り立つ。

**証明**

まず $D(W_1)=D(W_2)=0$ の場合、$W_1,W_2$ のいずれにも $X$ が存在しないので $W=W_1W_2$ にも $X$ は存在しない。したがって $D(W)=0$ であり、定義より $\bigl(B(W),A(W)\bigr)=\bigl(\mathrm{None},\mathrm{None}\bigr)$ である。

以下 $D(W)>0$ の場合を示す。$D(W_2)=0<D(W_1)$ なら $W$ の $X$ はすべて $W_1$ 由来であり、$\bigl(B^{\ast}(W),A^{\ast}(W)\bigr)$ はそのまま $\bigl(B^{\ast}(W_1),A^{\ast}(W_1)\bigr)$ である。$D(W_1)=0<D(W_2)$ なら $W$ の $X$ はすべて $W_2$ 由来で、各候補値は $S(W_1)$ だけ平行移動されるので

$$
\bigl(B^{\ast}(W),A^{\ast}(W)\bigr)
=\bigl(S(W_1)+B^{\ast}(W_2),A^{\ast}(W_2)\bigr)
$$

である。

$D(W_1),D(W_2)>0$ では

$$
L:=B^{\ast}(W_1),\qquad R:=S(W_1)+B^{\ast}(W_2)
$$

とおくと、$W$ の候補 prefix は $W_1$ 内候補（最大 $L$）か、$W_2$ 側候補に $S(W_1)$ を足したもの（最大 $R$）のいずれかである。ゆえに $B^{\ast}(W)=\max(L,R)$。最小達成添字は $L\ge R$ なら左側の $A^{\ast}(W_1)$、$L<R$ なら $D(W_1)+A^{\ast}(W_2)$ となる。同値 $L=R$ では左側を優先するので上式を得る。□

#### 補題（`MwfElem` の冪更新）

`floor_prod` では $x^k,y^k$ が現れるため、`MwfElem` の $k$ 乗は次の閉形式で与えられる。
定義（`MwfElem` の意味づけ）で導入した型 $\mathcal{T}_{\mathrm{mwf}}$ を用いて

$$
Z=(S,B,D,A)\in\mathcal{T}_{\mathrm{mwf}}
$$

とし、$D>0$ のとき

$$
(B,A)=\bigl(\mathrm{Some}(B^{\ast}),\mathrm{Some}(A^{\ast})\bigr)
$$

とおく。このとき

$$
Z^k=\bigl(kS,B_k,kD,A_k\bigr)
$$

$$
\bigl(B_k,A_k\bigr)=
\begin{cases}
\bigl(\mathrm{None},\mathrm{None}\bigr), & k=0\ \text{or}\ D=0,\\
\bigl(\mathrm{Some}((k-1)S+B^{\ast}),\mathrm{Some}((k-1)D+A^{\ast})\bigr), & k>0,\ D>0,\ S>0,\\
\bigl(\mathrm{Some}(B^{\ast}),\mathrm{Some}(A^{\ast})\bigr), & k>0,\ D>0,\ S\le 0.
\end{cases}
$$

が成り立つ。

**証明**

$Z^k$ は同一語の $k$ 連結に対応する。$k=0$ のときは空語に対応するので $kD=0$ かつ候補は存在せず、$\bigl(B_k,A_k\bigr)=\bigl(\mathrm{None},\mathrm{None}\bigr)$ である。また $D=0$ なら各ブロックに $X$ が存在しないため、任意の $k$ で同様に $\bigl(B_k,A_k\bigr)=\bigl(\mathrm{None},\mathrm{None}\bigr)$ となる。

以下 $k>0,\ D>0$ を考える。仮定より

$$
(B,A)=\bigl(\mathrm{Some}(B^{\ast}),\mathrm{Some}(A^{\ast})\bigr)
$$

であり、各ブロック先頭候補は

$$
B^{\ast},\ S+B^{\ast},\ 2S+B^{\ast},\dots,(k-1)S+B^{\ast}
$$

である。したがって $S>0$ なら末尾ブロックが最大で

$$
B_k^{\ast}=(k-1)S+B^{\ast},\qquad A_k^{\ast}=(k-1)D+A^{\ast},
$$

$S\le0$ なら先頭ブロックが最大で

$$
B_k^{\ast}=B^{\ast},\qquad A_k^{\ast}=A^{\ast}
$$

となる。同値時は左優先（最小添字）なのでこの分岐でよい。以上を $\mathrm{Some}(\cdot)$ で包めば主張を得る。□

この設計により「連結した列の prefix 最大値」を保つことができ、結果

$$
\mathtt{res.best}=\max_{0\le i<n}\phi(i),\qquad
\mathtt{res.arg}=\min\left\{i\in\mathbb{Z}_{\ge0}\ \middle|\ 0\le i<n,\ \phi(i)=\mathtt{res.best}\right\}
$$

を得る。実装では $n>0$ を前提にしているため、最終的に $\mathtt{res.dx}>0$ が成り立ち、$\mathtt{res.best}, \mathtt{res.arg}$ は有効値として読める。したがって `floor_prod` を 1 回呼ぶだけで$\max$ と最小の $\arg\max$ が同時に得られる。

**計算量**

モノイド積 1 回の計算量を $C$ とし、冪 $z^k$ を二分累乗で計算すると、`floor_prod` 全体は

$$
\mathcal{O}\!\left(C\bigl(\log n+\log m+\log a+\log b\bigr)\right)
$$

である。内訳は、$\mathtt{monoid\_pow}(y,\lfloor a/m\rfloor)$ の総和が$\mathcal{O}(\log a)$、$\mathtt{monoid\_pow}(y,\lfloor b/m\rfloor)$ が$\mathcal{O}(\log b)$、末尾側で現れる $x$ の冪が $\mathcal{O}(\log m)$、および停止時の $x^n$ が $\mathcal{O}(\log n)$ であることによる。ここで `MwfElem` を $C=\mathcal{O}(1)$ とすると、`mwf` 計算は対数時間で実行できる。さらに本稿の実装では `MwfElem.__pow__` を上記閉形式で実装しているため、単位コストモデルでは各べき計算自体は $\mathcal{O}(1)$ である。

本稿で用いる具体的な実装例は、実装節の[floor_prod に基づく mwf の実装](#floorprod-に基づく-mwf-の実装)で示す。

### x_min の計算量評価

**`mwf` の計算量**

前節[mwf の計算量](#mwf-の計算量)（計算量）で示したとおり、`step` の反復回数は

$$
\mathtt{stepBoundOfM}(m_0)=g(m_0)\le \left\lfloor\log_{\varphi}m_0\right\rfloor+2
$$

で抑えられる。本問題では除数側初期値は $m_0=A$ であるため、`mwf`（および閾値判定版 `mwf_leq`）1 回の評価はユークリッド互除法と同様に $\mathcal{O}(\log A)$ である。

**全体計算量**

二分探索の探索範囲は $u\in[0,U)$（$U=ABK+2$）なので、反復回数は $\mathcal{O}(\log U)=\mathcal{O}(\log(ABK+2))$ である。各反復で行う判定は `mwf_leq` 1 回（$\mathcal{O}(\log A)$）であるから、各テストケースの全体計算量は概ね

$$
\mathcal{O}\bigl(\log(ABK+2)\cdot\log A\bigr)
$$

となる（ただし積 $AB$ などの多倍長演算コストは別途考慮する）。

**$x_{\min}<X$ のみを判定する場合の計算量** $x_{\min}$ 自体ではなく、所与の $X\in\mathbb{Z}_{\ge0}$ に対して $x_{\min}(D,A,B,K)<X$ の真偽のみが必要な場合を考える。

ここでも解の存在と、$N:=\left\lceil\frac{X}{D}\right\rceil>0$ を仮定する。このとき $x_{\min}<X\iff u_{\min}<N$ であり、[補題（$x_{\min}$ が $X$ 未満であることの判定の帰着）](#補題（x_min-が-x-未満であることの判定の帰着）)のとおり

$$
x_{\min}<X\iff F(0,N)>BK
$$

で判定できる。したがって必要な計算は $F(0,N)$（あるいは閾値判定版 `mwf_leq`）の評価 1 回のみであり、（多倍長演算のコストを除けば）計算量は $\mathcal{O}(\log A)$ 程度となる。

### $10^{19}$ 除算補助関数の正当性

[背景説明](#背景説明)で述べた事例への応用として、固定除数 $D=10^{19}$ に対する商・剰余の仕様関数 `divmod_d19` を次で定める。

$$
\texttt{divmod\_d19}(x):=
\left(\left\lfloor\frac{x}{10^{19}}\right\rfloor,\ x\bmod 10^{19}\right)
$$

さらに、16 桁ごとに 3 つの部分へ分解する仕様関数 `divmod_d16` を次で定める。

$$
\texttt{divmod\_d16}(x):=
\left(\left\lfloor\frac{x}{10^{32}}\right\rfloor,\ \left\lfloor\frac{x\bmod 10^{32}}{10^{16}}\right\rfloor,\ x\bmod 10^{16}\right)
$$

**コード例**

`divmod_d19_125bit`, `divmod_d19_127bit`, `divmod_d19_128bit` は、想定した入力範囲において `divmod_d19` と同じ結果を返す補助関数である。  
`divmod_d16_128bit` は、想定した入力範囲において `divmod_d16` と同じ結果を返す補助関数である。以下に実装例を示す。

```python
def divmod_d19(x: int) -> tuple[int, int]:
    """仕様関数。`divmod(x, 10**19)` を実装する。"""
    return divmod(x, 10**19)

def divmod_d16(x: int) -> tuple[int, int, int]:
    """(x // 10**32, (x % 10**32) // 10**16, x % 10**16) を実装する仕様関数。"""
    return x // 10**32, (x % 10**32) // 10**16, x % 10**16

def divmod_d19_125bit(x: int) -> tuple[int, int]:
    """125bit 近似商 + 最大1回補正 で `divmod(x, 10**19)` を実行する補助関数。"""
    assert 0 <= x < 78312161395427422060000000000000000000
    d: int = 10**19
    a: int = 63
    b: int = 64
    m: int = (1 << (a + b)) // d  # m.bit_length() == 64
    q: int = (x >> a) * m >> b
    r: int = x - q * d
    if r >= d:
        q += 1
        r -= d
    assert (q, r) == divmod_d19(x)
    return q, r

def divmod_d19_127bit(x: int) -> tuple[int, int]:
    """127bit 近似商 + 最大2回補正 で `divmod(x, 10**19)` を実行する補助関数。"""
    assert 0 <= x < 1086673501021195308190000000000000000000
    d: int = 10**19
    a: int = 63
    b: int = 64
    m: int = (1 << (a + b)) // d  # m.bit_length() == 64
    q: int = (x >> a) * m >> b
    r: int = x - q * d
    if r >= d * 2:
        q += 2
        r -= d * 2
    elif r >= d:
        q += 1
        r -= d
    assert (q, r) == divmod_d19(x)
    return q, r

def divmod_d19_128bit(x: int) -> tuple[int, int]:
    """128bit 近似商 + 最大3回補正 で `divmod(x, 10**19)` を実行する補助関数。"""
    assert 0 <= x < 1164985662416622730250000000000000000000
    d: int = 10**19
    a: int = 64
    b: int = 64
    m: int = (1 << (a + b)) // d  # m.bit_length() == 65
    q: int = (x >> a) * m >> b
    r: int = x - q * d
    if r >= d * 3:
        q += 3
        r -= d * 3
    elif r >= d * 2:
        q += 2
        r -= d * 2
    elif r >= d:
        q += 1
        r -= d
    assert (q, r) == divmod_d19(x)
    return q, r

def divmod_d16_128bit(x: int) -> tuple[int, int, int]:
    """(x // 10**32, (x % 10**32) // 10**16, x % 10**16) を実装する補助関数。"""
    assert 0 <= x < 508484200000000000000000000000000000000
    d1: int = 10**32
    a1: int = 64
    b1: int = 64
    m1: int = (1 << (a1 + b1)) // d1
    d2: int = 10**16
    a2: int = 52
    b2: int = 64
    m2: int = (1 << (a2 + b2)) // d2
    q: int = (x >> a1) * m1 >> b1
    r: int = x - q * d1
    if r >= d1:
        q += 1
        r -= d1
    s: int = (r >> a2) * m2 >> b2
    t: int = r - s * d2
    if t >= d2:
        s += 1
        t -= d2
    assert (q, s, t) == (x // d1, (x % d1) // d2, (x % d1) % d2)
    return q, s, t
```

## 実装

### `mwf`

$\mathtt{mwf}(l,r,m,a,b,c,d)=\max_{x\in [l,r)}\left(ax+b\left\lfloor\frac{cx+d}{m}\right\rfloor\right)$を求める `mwf`（Max Weighted Floor of Linear）  [yukicoder no.3397](https://yukicoder.me/problems/no/3397) の Python 実装例を以下に示す。
```python
# -*- coding: utf-8 -*-
"""
solution for Max Weighted Floor of Linear
"""

def mwf(
    l: int, r: int, m: int,
    a: int, b: int, c: int, d: int
) -> int:
    """
    mwf(l,r,m,a,b,c,d) = max_{x in [l,r)} (a*x + b*floor((c*x + d)/m))
    """
    assert l < r and 0 < m
    n = r - l
    qd0, d = divmod(c * l + d, m)
    max_acc = sum_acc = a * l + b * qd0
    while True:
        qc, c = divmod(c, m)
        a += b * qc
        qd, d = divmod(d, m)
        sum_acc += b * qd
        y_max = (c * (n - 1) + d) // m
        max_acc = max(max_acc, sum_acc, sum_acc + a * (n - 1) + b * y_max)
        if (a <= 0 and b <= 0) or (a >= 0 and b >= 0) or y_max == 0:
            return max_acc
        if a < 0:
            sum_acc += a + b
        n, m, a, b, c, d = y_max, c, b, a, m, (m - d - 1)

if __name__ == '__main__':
    import sys
    T = int(sys.stdin.readline())
    for _ in range(T):
        N, M, A, B, C, D = map(int, sys.stdin.readline().split())
        max_val = mwf(0, N, M, A, B, C, D)
        print(max_val)
```

### `floor_prod` に基づく `mwf` の実装

理論的背景（`floor_prod` の定義、再帰更新、`mwf` への写像、計算量）は小節[floor_prod に基づく mwf](#floor_prod-に基づく-mwf)で述べた。以下では、$\arg\max$ も同時に求める`floor_prod` ベース実装例を示す。
ただし [yukicoder no.3397](https://yukicoder.me/problems/no/3397) では最大値のみを用いるため、求めた $\arg\max$ は実装上 `_argmax` として読み捨てている。

```python
# -*- coding: utf-8 -*-
"""
solution for Max Weighted Floor of Linear
"""

from __future__ import annotations
from dataclasses import dataclass

@dataclass(frozen=True)
class MwfElem:
    """
    Purpose: Represents a data structure for computing weighted maximum prefix sums.

    Fields:
      sum  : total accumulated sum
      best : maximum prefix value
      dx   : length / shift amount (used to compute arg)
      arg  : argmax (smallest arg if tie)
    """
    sum: int
    best: int = 0
    dx: int = 0
    arg: int = 0

    def __mul__(self, other: "MwfElem") -> "MwfElem":
        assert self.dx >= 0 and self.arg >= 0 and other.dx >= 0 and other.arg >= 0
        ssum = self.sum + other.sum
        if other.dx == 0:
            return MwfElem(ssum, self.best, self.dx, self.arg)
        cand_r = self.sum + other.best
        if self.dx == 0:
            return MwfElem(ssum, cand_r, other.dx, other.arg)
        sdx = self.dx + other.dx
        cand_l = self.best
        if cand_l >= cand_r:
            return MwfElem(ssum, cand_l, sdx, self.arg)
        return MwfElem(ssum, cand_r, sdx, self.dx + other.arg)

    def __pow__(self, k: int) -> "MwfElem":
        assert self.dx >= 0 and self.arg >= 0 and k >= 0
        ssum = self.sum * k
        if k == 0 or self.dx == 0:
            return MwfElem(ssum)
        sdx = self.dx * k
        if self.sum > 0:
            sbest = (k - 1) * self.sum + self.best
            sarg = (k - 1) * self.dx + self.arg
            return MwfElem(ssum, sbest, sdx, sarg)
        return MwfElem(ssum, self.best, sdx, self.arg)

def floor_prod(n: int, m: int, a: int, b: int, e: MwfElem, x: MwfElem, y: MwfElem):
    """
    Requires:
      - integers n,m,a,b with 0<=n, 0<m, 0<=a, 0<=b
      - x,y,e are elements of a monoid-like structure:
          * multiplication:  t1 * t2
          * power:           t ** k for k>=0
    """
    assert 0 <= n and 0 < m and 0 <= a and 0 <= b
    c = (a * n + b) // m
    pre, suf = e, e
    while True:
        p, a = divmod(a, m)
        x *= y ** p
        q, b = divmod(b, m)
        pre *= y ** q
        c -= p * n + q
        assert c == (a * n + b) // m
        if c == 0:
            return pre * (x ** n) * suf
        d = (m * c - b - 1) // a + 1
        assert d == n - ((a * n + b) % m) // a
        suf = y * (x ** (n - d)) * suf
        n, m, a, b, c, x, y = c - 1, a, m, m - b - 1 + a, d, y, x

def argmax_weighted_floor(n: int, m: int, a: int, b: int, c: int, d: int) -> tuple[int, int]:
    """
    argmwf(n,m,a,b,c,d) = (
      max_{0<=i<n} (a*i+b*floor((c*i+d)/m)),
      argmax (smallest i if tie)
    )
    """
    assert n > 0 and m > 0
    res: MwfElem = floor_prod(
        n, m, c, d,
        MwfElem(0),
        MwfElem(a, 0, 1, 0),
        MwfElem(b),
    )
    assert res.dx > 0
    return res.best, res.arg

if __name__ == '__main__':
    import sys
    T = int(sys.stdin.readline())
    for _ in range(T):
        N, M, A, B, C, D = map(int, sys.stdin.readline().split())
        max_val, _argmax = argmax_weighted_floor(N, M, A, B, C, D)
        print(max_val)
```

### `divapprox2`

`mwf` の閾値判定版 `mwf_leq` と、本稿の目的関数 $x_{\min}(D,A,B,K)$ を求める `divapprox2` （Accuracy of Integer Division Approximate Function 2）  [yukicoder no.3398](https://yukicoder.me/problems/no/3398) の Python 実装例を示す。[補題（最大安全長と最小解の一致）](#補題（最大安全長と最小解の一致）)のとおり、二分探索は[補題（最大安全長の表式）](#補題（最大安全長の表式）)の表式に従い「$F(0,\mathtt{lo})\le BK$（まだ閾値未達）を満たす最大の `lo`」を求める形で進める。また判定は、[補題（後半区間への判定移送）](#補題（後半区間への判定移送）)により区間最大値 $F(\mathtt{lo},\mathtt{mid})$ の閾値比較に帰着し、`mwf_leq` により「途中で $>BK$ が確定した時点で即座に打ち切る」形で高速化する。Python の整数は多倍長であるため、本問で必要となる $AB$ や探索上界 $U$ が 64-bit 整数で表現できる範囲を超える場合もそのまま扱える。
```python
# -*- coding: utf-8 -*-
"""
solution for Accuracy of Integer Division Approximate Function 2
"""
import math

def mwf_leq(
    t: int, l: int, r: int, m: int,
    a: int, b: int, c: int, d: int
) -> bool:
    """(max_{l <= x < r} a*x + b*floor((c*x + d)/m)) <= t"""
    assert l < r and 0 < m
    n = r - l
    qd0, d = divmod(c * l + d, m)
    sum_acc = a * l + b * qd0 - t
    if sum_acc > 0:
        return False
    while True:
        qc, c = divmod(c, m)
        a += b * qc
        qd, d = divmod(d, m)
        sum_acc += b * qd
        if sum_acc > 0:
            return False
        y_max = (c * (n - 1) + d) // m
        if (sum_acc + a * (n - 1) + b * y_max) > 0:
            return False
        if y_max == 0 or (a >= 0 and b >= 0) or (a <= 0 and b <= 0):
            return True
        if a < 0:
            sum_acc += a + b
        n, m, a, b, c, d = y_max, c, b, a, m, (m - d - 1)

def compute_xmin(d: int, a: int, b: int, k: int) -> int:
    """
    Compute xmin(D,A,B,K), return -1 if not exists
    xmin(D,A,B,K) = min{ x >= 0 | K < Δ(D,A,B,x) }
    Δ(D,A,B,x) = floor(x/D) - floor( floor(x/A) * floor(AB/D) / B )
    """
    assert d > 0 and a > 0 and b > 0 and k >= 0
    g, bk = math.gcd(d, a), b * k
    m, r = divmod(a * b, d)
    if r == 0 and d * k + g >= a:
        return -1
    if r == 0:
        # K+1 <= u_min < A/g
        lo = k + 1
        hi = a // g
    else:
        # max(floor((ABK-(A-g)M)/R)+1, K+1) <= u_min < floor(ABK/R)+2
        lo = max((a * b * k - (a - g) * m) // r + 1, k + 1)
        hi = (a * b * k) // r + 2
    while lo + 1 < hi:
        mid = (lo + hi) // 2
        if mwf_leq(bk, lo, mid, a, b, -m, d, 0):
            lo = mid
        else:
            hi = mid
    return d * lo

if __name__ == '__main__':
    import sys
    T = int(sys.stdin.readline())
    for _ in range(T):
        D, A, B, K = map(int, sys.stdin.readline().split())
        x = compute_xmin(D, A, B, K)
        print(x)
```

## 結論

主要な結論は次の通りである。最小解は $D$ の倍数に限られる（[補題（非倍数点での非増加）](#補題（非倍数点での非増加）)）ため$x=Du$ へ縮約でき、判定不等式は

$$
K<\Delta(D,A,B,Du)\iff BK< Bu-M\lfloor Du/A\rfloor\iff ABK<((Du)\bmod A)M+Ru
$$

と同値化できる（[補題（不等式の同値変形）](#補題（不等式の同値変形）)）。これにより探索範囲を有限化したうえで最大値判定に基づく二分探索を適用できる（[補題（最大安全長の表式）](#補題（最大安全長の表式）), [補題（後半区間への判定移送）](#補題（後半区間への判定移送）), [補題（最大安全長と最小解の一致）](#補題（最大安全長と最小解の一致）)）。結果として、各テストケースは概ね

$$
\mathcal{O}\bigl(\log(ABK+2)\cdot\log A\bigr)
$$

で解け、多倍長領域を含む入力にも適用可能である。
