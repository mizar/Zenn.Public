正の整数 $D$ を除数とする除算を行う場合、
$\displaystyle\frac{1}{D}$ を $\displaystyle\frac{M}{2^{S}}$ で近似し、
$0\leq n\leq N$ の範囲における全ての 整数 $n$ において、
$\displaystyle\left\lfloor\frac{n}{D}\right\rfloor=\left\lfloor\frac{M\times n}{2^{S}}\right\rfloor$
が成り立つような 非負整数 $M,S$ の組を前計算の段階で決定し、
乗算 + 右シフト演算(2の累乗数を除数とする整数除算の商を求める演算に相当) にて 除算 の代用とするような処理は、
コンパイラ等が行う除算最適化の手法のひとつとなっています。

これは、ハードウェアによる除算の実装は乗算に比べると高速化が難しく、
乗算 + 右シフト演算 による代用はハードウェアによる除算に比べると（コンパイル時最適化などで前計算のコストを無視できる状況であれば）実行時間を縮めやすいためです。

この 非負整数 $M,S$ の組は、例えば

$$
\begin{equation}
D\gt N,\quad M\times N \lt 2^S
\end{equation}
$$

もしくは

$$
\begin{equation}
M=\left\lceil\frac{2^S}{D}\right\rceil=\left\lfloor\frac{2^S-1}{D}\right\rfloor+1,\quad\left(\left(D\times M-2^S\right)\times N\right)\lt 2^S
\end{equation}
$$

もしくは

$$
\begin{equation}
D-1\leq N,\quad M=\left\lceil\frac{2^S}{D}\right\rceil=\left\lfloor\frac{2^S-1}{D}\right\rfloor+1,\quad\left(\left(D\times M-2^S\right)\times\left(D\times\left\lfloor\frac{N+1}{D}\right\rfloor-1\right)\right)\lt 2^S
\end{equation}
$$

のいずれかを満たすよう選べば良いので、必ずしも本問のように数え上げをする必要はありません。

条件によっては、 $0\leq M\leq N$ の範囲で適切な 非負整数 $M,S$ の組を見つける事ができない場合もあるため、その時は $0\leq M\leq\max(2N,\ 1)$ の範囲に広げる必要があります。
$0\leq M\leq \max(2N,\ 1)$ が許されるのであれば、例えば

$$
\begin{equation}
S=\left\lceil\log_2((D-1)N+1)\right\rceil=\left\lfloor\log_2\left(\max\left(2(D-1)N,\ 1\right)\right)\right\rfloor,\quad M=\left\lceil\frac{2^S}{D}\right\rceil=\left\lfloor\frac{2^S-1}{D}\right\rfloor+1
\end{equation}
$$

とすると $0\leq n\leq N$ の範囲における全ての 整数 $n$ において
$\displaystyle\left\lfloor\frac{n}{D}\right\rfloor=\left\lfloor\frac{M\times n}{2^{S}}\right\rfloor$
が成り立つ 非負整数 $M,S$ の組を得ることができます。

## コンパイラによる最適化の模倣

コンパイラの 64bit 除算最適化などの実装例では、 $2$ の累乗数での除算など特殊な場合を除き、
$\displaystyle 0\leq M=\left\lceil\frac{2^S}{D}\right\rceil\leq N=2^W-1$
として使えるマジックナンバーを探索し、どうしても見つからなければ
$\displaystyle 0\leq M'=\left\lceil\frac{2^S}{D}\right\rceil-2^W\lt 2^W$
の値をマジックナンバーとして使う、という事をしているようです。

- [(godbolt.org) コード例(C++): 除数 $998244353,\ 7$ での符号なし64bit整数除算の最適化](https://godbolt.org/z/6fbWGsEhr)
- [(godbolt.org) コード例(Rust): 除数 $998244353,\ 7$ での符号なし64bit整数除算の最適化](https://rust.godbolt.org/z/45GWzoT7K)
- 前提条件: 整数乗算の上位ワード $\lfloor(W\text{ bit 整数})\times (W\text{ bit 整数}) / 2^W\rfloor$ が計算できること、
$\displaystyle D,S,x\in\mathbb{Z},\ W=64,\ 0\leq x\lt 2^W,\ 1\leq D,\ 0\leq S,$ 
$\displaystyle M=\left\lceil\frac{2^S}{D}\right\rceil,\ \left(\left(D\times M-2^S\right)\times\left(2^W-1\right)\right)\lt 2^S$

```cpp
#include <cstdint>
#ifdef _MSC_VER
#include <intrin.h>
#endif
// C++: 乗算の結果128bitのうち上位64bitの値を返す
// u64mulh_shim(a, b) == floor(a * b / 2**64)
uint64_t u64mulh_shim(uint64_t a, uint64_t b) {
#ifdef _MSC_VER // msvc https://learn.microsoft.com/ja-jp/cpp/intrinsics/umulh?view=msvc-170
    return __umulh(a, b);
#else // gcc/clang use 128bit integer https://gcc.gnu.org/onlinedocs/gcc/_005f_005fint128.html
    return (unsigned __int128)a * (unsigned __int128)b >> 64;
#endif
}
```
```rust
// Rust: 乗算の結果128bitのうち上位64bitの値を返す
// u64mulh_shim(a, b) == floor(a * b / 2**64)
pub fn u64mulh_shim(a: u64, b: u64) -> u64 {
    ((a as u128) * (b as u128) >> 64) as u64
}
```
- マジックナンバー $M$ を使用:
$\displaystyle\left(0\leq M=\left\lceil\frac{2^S}{D}\right\rceil\lt 2^W\right)$
        - $\displaystyle\left\lfloor\frac{x}{D}\right\rfloor=\left\lfloor x\times\left\lceil\frac{2^S}{D}\right\rceil\times\frac{1}{2^S}\right\rfloor$
        - 除数 $998244353$ での符号なし64bit整数除算の最適化
```cpp
// C++: div998244353_shim(x) == floor(x / 998244353)
uint64_t div998244353_shim(uint64_t x) {
    // 0 <= x < (2**93 / ((-2**93) mod 998244353)) --> floor(ceil(2**93 / 998244353) * x / 2**93) == floor(x / 998244353)
    // ceil(2**93 / 998244353) == floor((2**93 - 1) / 998244353) + 1 == 9920937979283557439
    // floor(x / 998244353) == floor((x * 9920937979283557439) / 2**93) == floor(floor((x * 9920937979283557439) / 2**64) / 2**29)
    return u64mulh_shim(x, 9920937979283557439ULL) >> 29;
}
```
```rust
// Rust: div998244353_shim(x) == floor(x / 998244353)
pub fn div998244353_shim(x: u64) -> u64 {
    // 0 <= x < (2**93 / ((-2**93) mod 998244353)) --> floor(ceil(2**93 / 998244353) * x / 2**93) == floor(x / 998244353)
    // ceil(2**93 / 998244353) == floor((2**93 - 1) / 998244353) + 1 == 9920937979283557439
    // floor(x / 998244353) == floor((x * 9920937979283557439) / 2**93) == floor(floor((x * 9920937979283557439) / 2**64) / 2**29)
    u64mulh_shim(x, 9920937979283557439) >> 29
}
```
- マジックナンバー $M'=M-2^W$ を使用:
$\displaystyle\left(W\lt S,\quad 2^W\leq M=M'+2^W=\left\lceil\frac{2^S}{D}\right\rceil\lt 2^{(W+1)}\right)$
	- $\displaystyle y=\left\lfloor x\times\left\lceil\frac{2^S}{D}\right\rceil\times\frac{1}{2^W}\right\rfloor-x=\left\lfloor x\times\left(\left\lceil\frac{2^S}{D}\right\rceil-2^W\right)\times\frac{1}{2^W}\right\rfloor$
	- $\displaystyle\left\lfloor\frac{x}{D}\right\rfloor=\left\lfloor x\times\left\lceil\frac{2^S}{D}\right\rceil\times\frac{1}{2^S}\right\rfloor=\left\lfloor\frac{x+y}{2^{(S-W)}}\right\rfloor=\left\lfloor\left(\left\lfloor\frac{x-y}{2}\right\rfloor+y\right)\times\frac{1}{2^{(S-W-1)}}\right\rfloor$
	- $\displaystyle 2^W\leq M=M'+2^W=\left\lceil\frac{2^S}{D}\right\rceil\lt 2^{(W+1)}$ の場合、
	$x+y$ を直接 $W$ bit 整数型の上で計算するとオーバーフローする恐れがあること、
	$0\leq y\leq x\lt 2^W$ であること、より 
	$\displaystyle\left(\left\lfloor\frac{x-y}{2}\right\rfloor+y\right)$ のような計算が行われています。
	- 除数 $7$ での符号なし64bit整数除算の最適化
```cpp
// C++: div7_shim(x) == floor(x / 7)
uint64_t div7_shim(uint64_t x) {
    // 0 <= x < (2**67 / ((-2**67) mod 7)) --> floor(ceil(2**67 / 7) * x / 2**67) == floor(x / 7)
    // y := floor((ceil(2**67 / 7) - 2**64) * x / 2**64)
    // floor(x / 7) == floor((x + y) / 2**3) == floor((floor((x - y) / 2) + y) / 2**2)
    // ceil(2**67 / 7) - 2**64 == 2635249153387078803
    uint64_t y = u64mulh_shim(x, 2635249153387078803ULL);
    return (((x - y) >> 1) + y) >> 2;
}
```
```rust
// Rust: div7_shim(x) == floor(x / 7)
pub fn div7_shim(x: u64) -> u64 {
    // 0 <= x < (2**67 / ((-2**67) mod 7)) --> floor(ceil(2**67 / 7) * x / 2**67) == floor(x / 7)
    // y := floor((ceil(2**67 / 7) - 2**64) * x / 2**64)
    // floor(x / 7) == floor((x + y) / 2**3) == floor((floor((x - y) / 2) + y) / 2**2)
    // ceil(2**67 / 7) - 2**64 == 2635249153387078803
    let y = u64mulh_shim(x, 2635249153387078803);
    (((x - y) >> 1) + y) >> 2
}
```

## $D\gt N$ の場合の証明

[式 $(1)$ の証明]: $D\gt N$ の場合、 $\displaystyle \forall n\in\{x\in\mathbb{Z}\mid 0\leq x\lt D\},\ \left\lfloor\frac{n}{D}\right\rfloor=0$ より、

$$
\begin{equation}
\begin{split}
&0\leq M\times N\lt 2^S \\
\longleftrightarrow\quad&0\leq M\times \frac{N}{2^S} \lt 1 \\
\longrightarrow\quad&\forall n\in\{x\in\mathbb{Z}\mid 0\leq x\leq N\lt D\},\ \left\lfloor M\times \frac{n}{2^S}\right\rfloor=\left\lfloor\frac{n}{D}\right\rfloor=0\\
\end{split}
\end{equation}
$$

## $D-1\leq N$ の場合の証明の準備

[式 $(2)$ の証明]:

$$
\begin{equation}
\begin{split}
&D,S,N\in\mathbb{Z},\quad 1\leq D,\quad 0\leq N,\quad 0\leq S, \\
&0\leq\left(D\times\left\lceil\frac{2^S}{D}\right\rceil-2^S\right)\times N\lt 2^S
\end{split}
\end{equation}
$$

を前提とすると、
$0\leq x\lt 1,\ D\geq 1,\ D,n\in\mathbb{Z}\ \longrightarrow\ \displaystyle\left\lfloor\frac{n}{D}\right\rfloor=\left\lfloor\frac{n+x}{D}\right\rfloor$
という事を利用して、

$$
\begin{equation}
\begin{split}
&0\leq\left(D\times\left\lceil\frac{2^S}{D}\right\rceil-2^S\right)\times N\lt 2^S \\
\longleftrightarrow\quad& 0\leq\left(\left\lceil\frac{2^S}{D}\right\rceil-\frac{2^S}{D}\right)\times\frac{N}{2^S}\lt\frac{1}{D}\\
\longleftrightarrow\quad&\frac{N}{D}\leq\left\lceil\frac{2^S}{D}\right\rceil\times\frac{N}{2^S} =\frac{N}{D}+\left(\left\lceil\frac{2^S}{D}\right\rceil-\frac{2^S}{D}\right)\times\frac{N}{2^S}\lt\frac{N}{D}+\frac{1}{D} \\
\longrightarrow\quad&\forall n\in\{x\in\mathbb{Z}\mid 0\leq x\leq N\},\ \left\lfloor\left\lceil\frac{2^S}{D}\right\rceil\times\frac{n}{2^S}\right\rfloor=\left\lfloor\frac{n}{D}\right\rfloor\\
\end{split}
\end{equation}
$$

であることが言えます。

## $D-1\leq N$ の場合の証明

次に、 $D-1\leq N$ の場合に限定して考えます。 $n\in\mathbb{Z},\ 0\leq n\leq N$ とした場合に、

$$
\begin{equation}
\left\lfloor\frac{n}{D}\right\rfloor\ne\left\lfloor\frac{n}{D}+\frac{1}{D}\right\rfloor
\end{equation}
$$

が成り立つのは、 $n\equiv(D-1)\ (\mathrm{mod}\ D)$ の場合のみであり、そのような $n$ の最大値を $N'$ とおくと

$$
\begin{equation}
N'=D\times\left\lfloor\frac{N+1}{D}\right\rfloor-1
\end{equation}
$$

$$
\begin{equation}
\begin{split}
&0\leq\left(D\times\left\lceil\frac{2^S}{D}\right\rceil-2^S\right)\times N'\lt 2^S\\
\longleftrightarrow\quad&\frac{N'}{D}\leq\left\lceil\frac{2^S}{D}\right\rceil\times\frac{N'}{2^S}\lt\frac{N'}{D}+\frac{1}{D}\\
\longrightarrow\quad&\forall n'\in\{x'\in\mathbb{Z}\mid 0\leq x'\leq N'\},\ \left\lfloor\left\lceil\frac{2^S}{D}\right\rceil\times\frac{n'}{2^S}\right\rfloor=\left\lfloor\frac{n'}{D}\right\rfloor\\
\end{split}
\end{equation}
$$

となります。また、 $D,N\in\mathbb{Z},\quad 0\leq D-1\leq N,\quad N'\leq N,\quad N'\equiv(D-1)\ (\mathrm{mod}\ D)$ より $\max(N-D+1, D-1)\leq N'\leq N\leq 2N'$ が成り立ちます。

$$
\begin{equation}
\begin{cases}
D-1\leq N\leq 2D-2\quad &\longrightarrow\quad N'=D-1\leq N\leq 2N' \\
2D-2\lt N\quad &\longrightarrow\quad N/2\lt N-D+1\leq N'\leq N\lt 2N' \\
\end{cases}
\end{equation}
$$

この $N'\leq N\leq 2N'$ を用いると、

$$
\begin{equation}
\begin{split}
&0\leq\left(D\times\left\lceil\frac{2^S}{D}\right\rceil-2^S\right)\times N'\lt 2^S\\
\longrightarrow\quad&\frac{N'}{D}\leq\left\lceil\frac{2^S}{D}\right\rceil\times\frac{N'}{2^S}\lt\frac{N'}{D}+\frac{1}{D}\\
\longrightarrow\quad&\frac{N}{D}\leq\left\lceil\frac{2^S}{D}\right\rceil\times\frac{N}{2^S}\lt\frac{N}{D}+\frac{N/N'}{D}\leq\frac{N}{D}+\frac{2}{D}\\
\longrightarrow\quad&2^SN\leq\left\lceil\frac{2^S}{D}\right\rceil\times D\times N\leq 2^S(N+1)+(2^S-1)\lt 2^S(N+2)\\
\longrightarrow\quad&\left\lfloor N\right\rfloor\leq\left\lfloor\left\lceil\frac{2^S}{D}\right\rceil\times\frac{D\times N}{2^S}\right\rfloor\leq\left\lfloor N+1\right\rfloor=\left\lfloor N+1+\frac{2^S-1}{2^S}\right\rfloor\\
\longrightarrow\quad&\left\lfloor\frac{N}{D}\right\rfloor\leq\left\lfloor\left\lceil\frac{2^S}{D}\right\rceil\times\frac{N}{2^S}\right\rfloor\leq\left\lfloor\frac{N}{D}+\frac{1}{D}\right\rfloor\\
\end{split}
\end{equation}
$$

が成り立ちます。 

また、
$n'\in\mathbb{Z},\ N'\lt n'\leq N$ となる $n'$ は、
$n'\not\equiv(D-1)\ (\mathrm{mod}\ D)$ であるため、

$$
\begin{equation}
\forall n'\in\{x'\in\mathbb{Z}\mid N'\lt x'\leq N\},\ \left\lfloor\frac{n'}{D}\right\rfloor=\left\lfloor\left\lceil\frac{2^S}{D}\right\rceil\times\frac{n'}{2^S}\right\rfloor=\left\lfloor\frac{n'}{D}+\frac{1}{D}\right\rfloor
\end{equation}
$$

となる事が分かります。

[式 $(3)$ の証明]: 以上の結果、 式 $(6)$ ～ $(13)$ をまとめると、 $\displaystyle D,N,S\in\mathbb{Z},\ 0\leq D-1\leq N,\ 0\leq S,\ N'=D\times\left\lfloor\frac{N+1}{D}\right\rfloor-1$ において

$$
\begin{equation}
\begin{split}
&0\leq\left(D\times\left\lceil\frac{2^S}{D}\right\rceil-2^S\right)\times N'\lt 2^S \\
\longleftrightarrow\quad&0\leq\left(\left\lceil\frac{2^S}{D}\right\rceil-\frac{2^S}{D}\right)\times\frac{N'}{2^S}\lt\frac{1}{D} \\
\longleftrightarrow\quad&\frac{N'}{D}\leq\left\lceil\frac{2^S}{D}\right\rceil\times\frac{N'}{2^S} =\frac{N'}{D}+\left(\left\lceil\frac{2^S}{D}\right\rceil-\frac{2^S}{D}\right)\times\frac{N'}{2^S}\lt\frac{N'}{D}+\frac{1}{D} \\
\longleftrightarrow\quad&\frac{N}{D}\leq\left\lceil\frac{2^S}{D}\right\rceil\times\frac{N}{2^S} =\frac{N}{D}+\left(\left\lceil\frac{2^S}{D}\right\rceil-\frac{2^S}{D}\right)\times\frac{N}{2^S}\lt\frac{N}{D}+\frac{N/N'}{D} \\
\longrightarrow\quad&\forall n\in\{x\in\mathbb{Z}\mid 0\leq x\leq N\},\ \left\lfloor\left\lceil\frac{2^S}{D}\right\rceil\times \frac{n}{2^S}\right\rfloor=\left\lfloor\frac{n}{D}\right\rfloor\\
\end{split}
\end{equation}
$$

## 式 $(4)$ の説明

まずは、 式 $(4)$ によって $\displaystyle\forall n\in\{x\in\mathbb{Z}\mid 0\leq x\leq N\},\ \left\lfloor\left\lceil\frac{2^S}{D}\right\rceil\times\frac{n}{2^S}\right\rfloor=\left\lfloor\frac{n}{D}\right\rfloor$ が成り立つ事を説明します。

$D,N\in\mathbb{Z},\ 1\leq D,\ 0\leq N$ の時に $S=\lceil\log_2((D-1)N+1)\rceil,\ M=\lceil 2^S/D\rceil$ とした場合、

$$
\begin{equation}
\begin{split}
&S=\lceil\log_2((D-1)N+1)\rceil \\
\longrightarrow\quad&2^{S-1}\lt(D-1)N+1\leq 2^S\leq 2(D-1)N+1\lt 2(D-1)N+2 \\
\longrightarrow\quad&2^{S-1}-1\lt(D-1)N\lt 2^S\leq 2(D-1)N+1\lt 2(D-1)N+2 \\
\end{split}
\end{equation}
$$

となります。また、除算の性質より

$$
\begin{equation}
0\leq\left(D\times\left\lceil\frac{2^S}{D}\right\rceil-2^S\right)\leq D-1\lt D
\end{equation}
$$

であるため、 式 $(15),(16)$ より

$$
\begin{equation}
0\leq\left(D\times\left\lceil\frac{2^S}{D}\right\rceil-2^S\right)N\leq(D-1)N\lt 2^S
\end{equation}
$$

となり、これは 式 $(2),(6)$ の前提を満たし、
$\displaystyle\forall n\in\{x\in\mathbb{Z}\mid 0\leq x\leq N\},\ \left\lfloor\left\lceil\frac{2^S}{D}\right\rceil\times\frac{n}{2^S}\right\rfloor=\left\lfloor\frac{n}{D}\right\rfloor$
が成り立ちます。

次に、 式 $(4)$ が $\displaystyle 0\leq M=\left\lceil\frac{2^S}{D}\right\rceil\leq\max(2N,1)$ を満たすことを説明します。

まずは、 $N=0$ における 式 $(4)$ は $S=0,\ M=1$ となり、 $0\leq M\leq\max(2N,1)$ となります。

次に、 $1\leq N$ であれば $0\leq\lfloor(2N-1)/D\rfloor$ が成り立つため、 式 $(15)$ より

$$
\begin{equation}
\begin{split}
&1\leq N,\quad S=\lceil\log_2((D-1)N+1)\rceil \\
\longrightarrow\quad&1\leq N,\quad 0\leq\frac{D-1}{D}N\lt\frac{2^S}{D}\leq M=\left\lceil\frac{2^S}{D}\right\rceil\leq\left\lceil\frac{2(D-1)N+1}{D}\right\rceil=2N-\left\lfloor\frac{2N-1}{D}\right\rfloor \\
\longrightarrow\quad&1\leq N,\quad 0\leq\frac{D-1}{D}N\lt M=\left\lceil\frac{2^S}{D}\right\rceil\leq 2N\\
\end{split}
\end{equation}
$$

となり、 $0\leq M\leq 2N$ 、つまり $0\leq M\leq\max(2N,1)$ が成り立つことを示しています。
